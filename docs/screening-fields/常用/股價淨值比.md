# 股價淨值比 – （選股欄位） <kbd>常用</kbd> <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 股價淨值比 |
| 欄位分類 | 常用、價格 |
| 代碼 | `PBRatio`<br>`PB` |
| 單位 | 倍 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月、季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 計算公式：收盤價／每股淨值。  

```pascal
Value1 = GetField("股價淨值比");
Value1 = GetField("PBRatio");
Value1 = GetField("PB");
```

---

## 說明
計算公式：收盤價／每股淨值。

週頻率以上為統計期間的期底值。
