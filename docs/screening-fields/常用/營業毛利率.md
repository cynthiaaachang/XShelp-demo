# 營業毛利率 – （選股欄位） <kbd>常用</kbd> <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 營業毛利率 |
| 欄位分類 | 常用、財務 |
| 代碼 | `GrossMargin` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 財務比率表上的「營業毛利率」欄位。  
> 提供單季與年度資料。  
> 不提供此欄位的公司，輸出值會以零表示。  
> **單位是％，例如：營業毛利率為30%，則回傳30。**  

```pascal
Value1 = GetField("營業毛利率");
Value1 = GetField("GrossMargin");
```
