# 股東權益報酬率 – （選股欄位） <kbd>常用</kbd> <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 股東權益報酬率 |
| 欄位分類 | 常用、財務 |
| 代碼 | `ROE`<br>`ReturnOnEquity` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 台股，是財務比率表的「ROE(A)─稅後」項目。  
> 美股，是財務比率表上的「股東權益報酬率」項目。  
> 提供單季與年度資料。  
> **單位為%，例如：股東權益報酬率為30%，則回傳30。**  

```pascal
Value1 = GetField("股東權益報酬率");
Value1 = GetField("ROE");
Value1 = GetField("ReturnOnEquity");
```
