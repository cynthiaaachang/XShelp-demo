# 收盤價 – （選股欄位） <kbd>常用</kbd> <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 收盤價 |
| 欄位分類 | 常用、價格 |
| 代碼 | `Close`<br>`c`<br>`close` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月、還原日、還原週、還原月、季、半年、年 |
| 支援商品 | 台股、香港股票、大陸股票、美(股票) |

## 語法
> K棒資料的收盤價(最後一個成交價位)。  

```pascal
Value1 = GetField("收盤價");
Value1 = GetField("Close");
Value1 = c
Value1 = close
```
