# 月營收 – （選股欄位） <kbd>常用</kbd> <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 月營收 |
| 欄位分類 | 常用、基本 |
| 代碼 | `Sales` |
| 單位 | 億 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 台股上市櫃公司每月公告的營業收入數值。  

```pascal
Value1 = GetField("月營收", "M");
Value1 = GetField("Sales", "M");
```

---

## 說明
台股上市櫃公司每月公告的營業收入數值。
欄位更新日期依照各家公司公佈的時間點而異。
依交易所規定，每月10號前需公佈上月營收。若截止日遇到假日，可順延至上班日再公布。
