## 語法
> 當期股價的漲跌幅。單位是％。  

```pascal
Value1 = GetField("漲跌幅");
Value1 = GetField("Change(％)");
```

---

## 說明
當期股價的漲跌幅。單位是％。

漲跌幅計算方式為(當期收盤價 - 當期參考價)／當期參考價  * 100%。

週頻率以上為區間統計值。
