# 鉅額交易量 – （選股欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 鉅額交易量 |
| 欄位分類 | 量能 |
| 代碼 | `BlockTrade` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 台股 |

## 語法
> 鉅額交易的成交量加總。  

```pascal
Value1 = GetField("鉅額交易量");
Value1 = GetField("BlockTrade");
```

---

## 說明
鉅額交易的成交量加總。

週頻率以上為區間累計值。
