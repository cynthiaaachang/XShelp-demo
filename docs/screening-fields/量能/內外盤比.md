# 內外盤比 – （選股欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 內外盤比 |
| 欄位分類 | 量能 |
| 代碼 | `BidAskRatio` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股 |

## 語法
> 計算公式  : 內盤量 / 外盤量 * 100%  
> 數值越大表示「賣方」力道越強勁。  

```pascal
Value1 = GetField("內外盤比", "D");
Value1 = GetField("BidAskRatio", "D");
```
