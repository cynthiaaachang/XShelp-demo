# 盤後量 – （選股欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 盤後量 |
| 欄位分類 | 量能 |
| 代碼 | `OffHourVolume` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 台股 |

## 語法
> 盤後定價交易區間的成交張數。  

```pascal
Value1 = GetField("盤後量");
Value1 = GetField("OffHourVolume");
```
