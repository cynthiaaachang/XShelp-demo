# 開盤量 – （選股欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 開盤量 |
| 欄位分類 | 量能 |
| 代碼 | `VolumeAtOpen` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股 |

## 語法
> 每天第一筆成交的成交量。  

```pascal
Value1 = GetField("開盤量", "D");
Value1 = GetField("VolumeAtOpen", "D");
```

## 說明
每天第一筆成交的成交量。
