# 開盤委買 – （選股欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 開盤委買 |
| 欄位分類 | 量能 |
| 代碼 | `BidAtOpen`<br>`DayOpenBid` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股、類股指數 |

## 語法
> 開盤時委託買進的數量。  

```pascal
Value1 = GetField("開盤委買", "D");
Value1 = GetField("BidAtOpen", "D");
Value1 = GetField("DayOpenBid", "D");
```

---

## 說明
開盤時委託買進的數量。

台股的數值是開盤第一筆成交前五檔委買的數量加總。

類股商品（細產業指標、主題指標）的計算方式則是成分股的開盤委買的加總。

大盤/期貨/選擇權的數值則是交易所公布的委買委賣資料的第一筆數值。大盤的數值會扣除權證。
