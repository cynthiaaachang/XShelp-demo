# 外盤成交次數 – （選股欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 外盤成交次數 |
| 欄位分類 | 量能 |
| 代碼 | `TotalOutTicks` |
| 單位 | 次 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股、類股指數、香港股票、大陸股票 |

## 語法
> 這一筆K棒的所有成交內以外盤成交的次數。  

```pascal
Value1 = GetField("外盤成交次數", "D");
Value1 = GetField("TotalOutTicks", "D");
```

---

## 說明
這一筆K棒的所有成交內以外盤成交的次數。
