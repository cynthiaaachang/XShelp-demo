# 總成交筆數 – （選股欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 總成交筆數 |
| 欄位分類 | 量能 |
| 代碼 | `TotalTransaction` |
| 單位 | 筆 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月、季、年 |
| 支援商品 | 台股 |

## 語法
> 交易時段中的成交總筆數。  

```pascal
Value1 = GetField("總成交筆數");
Value1 = GetField("TotalTransaction");
```

---

## 說明
交易時段中的成交總筆數。

週頻率以上為區間累計值。

本統計資訊含一般、零股、盤後定價，不含鉅額、拍賣、標購。
