# 佔全市場成交量比 – （選股欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 佔全市場成交量比 |
| 欄位分類 | 量能 |
| 代碼 | `WeightOfMarket` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股 |

## 語法
> 計算公式：個股成交值 / (加權指數成交值+櫃買指數成交值) * 100%。  
> **單位為%，例如比值為1.5%時，回傳值為1.5。**  

```pascal
Value1 = GetField("佔全市場成交量比", "D");
Value1 = GetField("WeightOfMarket", "D");
```
