# 內盤均量 – （選股欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 內盤均量 |
| 欄位分類 | 量能 |
| 代碼 | `BidAvg` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股 |

## 語法
> 計算公式：內盤成交量／內盤成交次數。  

```pascal
Value1 = GetField("內盤均量", "D");
Value1 = GetField("BidAvg", "D");
```
