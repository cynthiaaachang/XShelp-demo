# 週平均收益率 – （選股欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 週平均收益率 |
| 欄位分類 | 價格 |
| 代碼 | `AvgReturnW` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股 |

## 語法
> 最近5日的平均賺賠比例。  
> **計算公式 = 近5日漲幅 / 5，其中近5日漲幅 = (收盤價 - 5日前收盤價) / 5日前收盤價 * 100%。**  

```pascal
Value1 = GetField("週平均收益率", "D");
Value1 = GetField("AvgReturnW", "D");
```

---

## 說明
最近5日的平均賺賠比例。

計算公式 = 近5日漲幅 / 5，其中近5日漲幅 = (收盤價 - 5日前收盤價) / 5日前收盤價 * 100%。

使用還原資料計算。
