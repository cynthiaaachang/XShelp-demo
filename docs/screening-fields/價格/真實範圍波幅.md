# 真實範圍波幅 – （選股欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 真實範圍波幅 |
| 欄位分類 | 價格 |
| 代碼 | `TR％` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月、季、年 |
| 支援商品 | 台股 |

## 語法
> **真實範圍(TrueRange) / 昨收價 * 100%。單位為%，假如真實範圍是5，昨收價是50，則真實範圍波幅為10%，回傳值為10。**  

```pascal
Value1 = GetField("真實範圍波幅");
Value1 = GetField("TR％");
```
