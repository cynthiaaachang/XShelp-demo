# Treynor – （選股欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | Treynor |
| 欄位分類 | 價格 |
| 代碼 | `Treynor` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股、16.US.E |

## 語法
> Treynor指標衡量投資相對於系統性風險的回報率。  

```pascal
Value1 = GetField("Treynor", "D");
Value1 = GetField("Treynor", "D");
```

---

## 說明
Treynor指標衡量投資相對於系統性風險的回報率。

計算公式為: (商品過去一個月的平均報酬率 - 無風險利率) / 商品的Beta值，用來衡量每單位系統風險所獲得的超額報酬。
