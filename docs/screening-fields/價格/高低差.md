# 高低差 – （選股欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 高低差 |
| 欄位分類 | 價格 |
| 代碼 | `Range` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月、季、年 |
| 支援商品 | 台股 |

## 語法
> 當期的最高價 － 當期的最低價。  

```pascal
Value1 = GetField("高低差");
Value1 = GetField("Range");
```
