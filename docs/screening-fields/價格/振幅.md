# 振幅 – （選股欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 振幅 |
| 欄位分類 | 價格 |
| 代碼 | `Range(％)` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月、季、年 |
| 支援商品 | 台股 |

## 語法
> 當期股價的振幅。單位為%。  

```pascal
Value1 = GetField("振幅");
Value1 = GetField("Range(％)");
```

---

## 說明
當期股價的振幅。單位為%。

振幅計算公式 = (當期最高價 - 當期最低價) * 100 / 參考價%。

週頻率以上為區間統計值。
