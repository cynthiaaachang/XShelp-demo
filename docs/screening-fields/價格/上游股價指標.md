# 上游股價指標 – （選股欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 上游股價指標 |
| 欄位分類 | 價格 |
| 代碼 | `UpStreamPriceIndex` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股 |

## 語法
> 利用供應商的股價波動計算股價指數。  

```pascal
Value1 = GetField("上游股價指標", "D");
Value1 = GetField("UpStreamPriceIndex", "D");
```

---

## 說明
利用供應商的股價波動計算股價指數。

從2004年1月1日為基期100後逐日計算。

上游股價指標的計算有包含國外股票，詳細的成份內容可以參考個股的"營運結構"。
