# 月平均收益率 – （選股欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 月平均收益率 |
| 欄位分類 | 價格 |
| 代碼 | `AvgReturnM` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股 |

## 語法
> 最近21日的平均賺賠比例。  

```pascal
Value1 = GetField("月平均收益率", "D");
Value1 = GetField("AvgReturnM", "D");
```

---

## 說明
最近21日的平均賺賠比例。

計算公式 = 近21日漲幅 / 21，其中近21日漲幅 = (收盤價 - 21日前收盤價) / 21日前收盤價 * 100%。

使用還原資料計算。
