# 標準差 – （選股欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 標準差 |
| 欄位分類 | 價格 |
| 代碼 | `StdDev` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股 |

## 語法
> 使用一年的每日漲跌幅，計算標準差數值，用來衡量股票報酬率的波動性。數值越小表示波動性越低。  

```pascal
Value1 = GetField("標準差", "D");
Value1 = GetField("StdDev", "D");
```
