# 最後過戶日期 – （選股欄位） <kbd>事件日期</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 最後過戶日期 |
| 欄位分類 | 事件日期 |
| 代碼 | `LastTransferDay` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 語法
> 最近一次除權息最後過戶日的日期。  
> 欄位格式為西元年月日，例如20230618。  

```pascal
Value1 = GetField("最後過戶日期");
Value1 = GetField("LastTransferDay");
```

---

## 說明
最新一期的除權息最後過戶日日期，以西元年月日來表示，當日期為2023年6月18日，回傳值為20230618。
會依計算日期取得最新的除權息最後過戶日期。
資料來源：TEJ台灣經濟新報。
