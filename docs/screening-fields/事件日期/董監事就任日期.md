# 董監事就任日期 – （選股欄位） <kbd>事件日期</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 董監事就任日期 |
| 欄位分類 | 事件日期 |
| 代碼 | `DateOfBoardOfDirectorInaugurate` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 語法
> 最新一期董監事就任日期。  
> 欄位格式為西元年月日，例如20100101。  

```pascal
Value1 = GetField("董監事就任日期");
Value1 = GetField("DateOfBoardOfDirectorInaugurate");
```
