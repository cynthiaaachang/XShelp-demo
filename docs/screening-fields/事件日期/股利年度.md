# 股利年度 – （選股欄位） <kbd>事件日期</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 股利年度 |
| 欄位分類 | 事件日期 |
| 代碼 | `DividendYear` |
| 單位 | 年 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 年 |
| 支援商品 | 台(股票)、美(股票)、16.US.E |

## 語法
> **回傳完整的股利年度。**  
> 欄位格式為西元年，例如2018。  

```pascal
Value1 = GetField("股利年度", "Y");
Value1 = GetField("DividendYear", "Y");
```
