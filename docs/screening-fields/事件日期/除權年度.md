# 除權年度 – （選股欄位） <kbd>事件日期</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 除權年度 |
| 欄位分類 | 事件日期 |
| 代碼 | `ExRightYear` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台(股票)、美(股票) |

## 語法
> 最近一次公佈的除權資料所對應的股利所屬年度。  
> 欄位格式為西元年，例如2015。  

```pascal
Value1 = GetField("除權年度");
Value1 = GetField("ExRightYear");
```

---

## 說明
最近一次公佈的除權資料所對應的股利所屬年度。

欄位格式為西元年，例如2015。

可以檢查GetFieldDate("除權年度")是否等於0來判斷是否有資料。
