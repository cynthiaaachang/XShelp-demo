# 股東會日期 – （選股欄位） <kbd>事件日期</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 股東會日期 |
| 欄位分類 | 事件日期 |
| 代碼 | `DateofMeetingofShareHolders` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 語法
> 最近一次股東會的日期。  
> 欄位格式為西元年月日，例如20221101。  

```pascal
Value1 = GetField("股東會日期");
Value1 = GetField("DateofMeetingofShareHolders");
```
