# 融券最後回補日 – （選股欄位） <kbd>事件日期</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 融券最後回補日 |
| 欄位分類 | 事件日期 |
| 代碼 | `ShortSaleFinalRecallDate` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 語法
> 最近一次融券最後回補日的日期。  
> 欄位格式為西元年月日，例如20221101。  

```pascal
Value1 = GetField("融券最後回補日");
Value1 = GetField("ShortSaleFinalRecallDate");
```

---

## 說明
最近一次融券最後回補日的日期。

欄位格式為西元年月日，例如20221101。

系統依照最近一次公布的股東會，除權息，現增，或是減資，回傳最接近的融券最後回補日。
