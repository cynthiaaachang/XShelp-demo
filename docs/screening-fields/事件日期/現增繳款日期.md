# 現增繳款日期 – （選股欄位） <kbd>事件日期</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 現增繳款日期 |
| 欄位分類 | 事件日期 |
| 代碼 | `DateofCashIncrement` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 語法
> 最近一次現金增資的最後繳款日。  
> 欄位格式為西元年月日，例如20221101。  

```pascal
Value1 = GetField("現增繳款日期");
Value1 = GetField("DateofCashIncrement");
```

---

## 說明
最近一次現金增資的最後繳款日。

欄位格式為西元年月日，例如20221101。

可以檢查GetFieldDate("現增繳款日期")是否等於0來判斷是否有現金增資的資料。
