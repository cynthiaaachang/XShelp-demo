# 減資比例 – （選股欄位） <kbd>事件日期</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 減資比例 |
| 欄位分類 | 事件日期 |
| 代碼 | `CapitalReductionRatio` |
| 單位 | % |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 語法
> 最近一次減資的比例。  
> 單位是％，例如20代表減資20%。  

```pascal
Value1 = GetField("減資比例");
Value1 = GetField("CapitalReductionRatio");
```

---

## 說明
最近一次減資的比例。

單位是％，例如20代表減資20%。

可以檢查GetFieldDate("減資比例")是否等於0來判斷是否有減資的資料。
