# 日期 – （選股欄位） <kbd>事件日期</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 日期 |
| 欄位分類 | 事件日期 |
| 代碼 | `Date`<br>`date` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月、還原日、還原週、還原月、季、半年、年 |
| 支援商品 | 全部 |

## 語法
> K棒資料的日期。格式為西元年月日，例如20221101。  

```pascal
Value1 = GetField("日期");
Value1 = GetField("Date");
Value1 = date
```
