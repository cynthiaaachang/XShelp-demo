# 累計每股盈餘(發佈值) – （選股欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 累計每股盈餘(發佈值) |
| 欄位分類 | 其他 |
| 代碼 | `BasicEPSAccumulatedAnnouncement` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 季 |
| 支援商品 | 台股 |

## 語法
> 公司於公開資訊觀測站所發佈重訊：  
> 1月1日累計至本期止基本每股盈餘(損失)  

```pascal
Value1 = GetField("累計每股盈餘(發佈值)", "Q");
Value1 = GetField("BasicEPSAccumulatedAnnouncement", "Q");
```
