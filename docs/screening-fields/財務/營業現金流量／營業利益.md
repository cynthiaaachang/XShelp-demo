# 營業現金流量／營業利益 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 營業現金流量／營業利益 |
| 欄位分類 | 財務 |
| 代碼 | `CFO／OperatingIncome` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 計算公式: 營業現金流量／營業利益*100%。  
> 提供單季與年度資料。  
> **單位為%，例如：數值為25%，則回傳25。**  

```pascal
Value1 = GetField("營業現金流量／營業利益");
Value1 = GetField("CFO／OperatingIncome");
```
