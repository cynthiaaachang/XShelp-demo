# 因子_日週轉率流動性 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 因子_日週轉率流動性 |
| 欄位分類 | 財務 |
| 代碼 | `Factor_TURNOVER_AVG`<br>`因子_流動性因子(以平均日週轉率為基準)`<br>`Factor_Trading volume Factor` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 過去一個月的日週轉率平均值。  

```pascal
Value1 = GetField("因子_日週轉率流動性", "M");
Value1 = GetField("Factor_TURNOVER_AVG", "M");
Value1 = GetField("因子_流動性因子(以平均日週轉率為基準)", "M");
Value1 = GetField("Factor_Trading volume Factor", "M");
```

---

## 說明
過去一個月的日週轉率平均值。
日週轉率=日成交量/發行量。
