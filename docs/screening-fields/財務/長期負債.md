# 長期負債 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 長期負債 |
| 欄位分類 | 財務 |
| 代碼 | `Long－TermLiabilities`<br>`非流動負債` |
| 單位 | 台股：百萬。美股：百萬(美元) |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 台股:「資產負債表」中「非流動負債」項目。  
> 美股:「資產負債表」中「非流動負債」項內「長期借款」項目。  
> 不提供此欄位的公司，輸出值會以零表示。  

```pascal
Value1 = GetField("長期負債");
Value1 = GetField("Long－TermLiabilities");
Value1 = GetField("非流動負債");
```

---

## 說明
台股:「資產負債表」中「非流動負債」項目。
美股:「資產負債表」中「非流動負債」項內「長期借款」項目。
不提供此欄位的公司，輸出值會以零表示。
