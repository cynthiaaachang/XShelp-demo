# 流動負債 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 流動負債 |
| 欄位分類 | 財務 |
| 代碼 | `TotalCurrentLiabilities` |
| 單位 | 台股：百萬。美股：百萬(美元) |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 「資產負債表」中「流動負債」項目。  
> 台股中金融、壽險、產險股不適用此欄位。  

```pascal
Value1 = GetField("流動負債");
Value1 = GetField("TotalCurrentLiabilities");
```

---

## 說明
「資產負債表」中「流動負債」項目。凡將於一年或一營業週期內（以較長者為準）以流動資產償還或再轉為流動負債皆屬之。
台股中金融、壽險、產險股不適用此欄位。
