# 流動比率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 流動比率 |
| 欄位分類 | 財務 |
| 代碼 | `CurrentRatio` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 財務比率表上的「流動比率」欄位。  
> 提供季度與年度資料。  
> **單位為%，例如：流動比率為250%，則回傳250。**  
> 不提供此欄位的公司，輸出值會以零表示。  

```pascal
Value1 = GetField("流動比率");
Value1 = GetField("CurrentRatio");
```
