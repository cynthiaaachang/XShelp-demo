# 1年夏普指數 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 1年夏普指數 |
| 欄位分類 | 財務 |
| 代碼 | `SHARPE_1Y` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> Sharpe指標衡量投資相對於其風險所帶來的超額回報率，即投資回報超過無風險利率部分的表現。  

```pascal
Value1 = GetField("1年夏普指數", "M");
Value1 = GetField("SHARPE_1Y", "M");
```

---

## 說明
Sharpe指標衡量投資相對於其風險所帶來的超額回報率，即投資回報超過無風險利率部分的表現。
計算方式為：(商品過去一年的平均月報酬率-無風險利率)/過去一年的月報酬率的標準差，用來衡量每單位風險所獲得的超額報酬。
