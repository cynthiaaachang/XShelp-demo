# 年報酬率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 年報酬率 |
| 欄位分類 | 財務 |
| 代碼 | `YearlyReturnRate` |
| 單位 | % |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股、16.US.E |

## 語法
> 僅支援台/美ETF商品.  
> 統計近N年報酬率, 計算方式包含配息, 以及配息再投入.  
> 欄位提供參數, 可以傳入年度, 支援1,3,5,10，預設是 1。  
> (如欲計算股票的報酬率, 可以使用還原股價直接計算, 系統會提供內建函數)  

```pascal
Value1 = GetField("年報酬率", param:=1);
Value1 = GetField("YearlyReturnRate", param:=1);
```
