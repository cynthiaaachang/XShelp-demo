# 因子分數_彩券型需求 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 因子分數_彩券型需求 |
| 欄位分類 | 財務 |
| 代碼 | `FactorScore_MAX`<br>`因子分數_彩券型需求因子`<br>`FactorScore_Lottery Demand Factor` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |

## 語法
> 彩券型需求因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。  

```pascal

Value1 = GetField("FactorScore_MAX", "M");
Value1 = GetField("因子分數_彩券型需求因子", "M");
Value1 = GetField("FactorScore_Lottery Demand Factor", "M");
```
