# 股利收入 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 股利收入 |
| 欄位分類 | 財務 |
| 代碼 | `InvestmentIncome`<br>`投資收入／股利收入` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「損益表」中「股利收入」項目。  
> 壽險、產險、證券股不適用此欄位。  

```pascal
Value1 = GetField("股利收入");
Value1 = GetField("InvestmentIncome");
Value1 = GetField("投資收入／股利收入");
```

---

## 說明
「損益表」中「股利收入」項目。凡非以投資為營業項目者，其採用權益法之投資以外之金融資產獲配之現金股利屬之。
壽險、產險、證券股不適用此欄位。
