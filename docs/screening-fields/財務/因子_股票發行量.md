# 因子_股票發行量 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 因子_股票發行量 |
| 欄位分類 | 財務 |
| 代碼 | `Factor_ISSUE`<br>`因子_股票發行量因子`<br>`Factor_Share issuance Factor` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 自然對數(當期股票面額/上季股票面額/前一季流通在外股數)。  

```pascal
Value1 = GetField("因子_股票發行量", "M");
Value1 = GetField("Factor_ISSUE", "M");
Value1 = GetField("因子_股票發行量因子", "M");
Value1 = GetField("Factor_Share issuance Factor", "M");
```
