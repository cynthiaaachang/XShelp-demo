# 流動資產 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 流動資產 |
| 欄位分類 | 財務 |
| 代碼 | `TotalCurrentAssets` |
| 單位 | 台股：百萬。美股：百萬(美元) |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 「資產負債表」中「流動資產」項目。  
> 台股中金融、壽險、產險股不適用此欄位。  

```pascal
Value1 = GetField("流動資產");
Value1 = GetField("TotalCurrentAssets");
```

---

## 說明
「資產負債表」中「流動資產」項目。凡現金及其他將於一年或一營業週期內（以較長者為準）變現、出售或耗用之資產皆屬之。
台股中金融、壽險、產險股不適用此欄位。
