| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 因子_成交值流動性 |
| 欄位分類 | 財務 |
| 代碼 | `Factor_DVOL`<br>`因子_流動性因子(以成交值為基準)`<br>`Factor_Trading volume Factor` |
