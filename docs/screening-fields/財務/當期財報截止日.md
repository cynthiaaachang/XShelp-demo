# 當期財報截止日 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 當期財報截止日 |
| 欄位分類 | 財務 |
| 代碼 | `PeriodEndDate` |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 美(股票) |

## 語法
> **回傳每個公司的財報截止日，格式為YYYYMMDD.**  
> 美股股票才支援此欄位。  

```pascal
Value1 = GetField("當期財報截止日");
Value1 = GetField("PeriodEndDate");
```

---

## 說明
回傳每個公司的財報截止日，格式為YYYYMMDD.
美股股票才支援此欄位。
