# 資產總額 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 資產總額 |
| 欄位分類 | 財務 |
| 代碼 | `TotalAssets` |
| 單位 | 台股：百萬。美股：百萬(美元) |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 「資產負債表」財務報表項目。  

```pascal
Value1 = GetField("資產總額");
Value1 = GetField("TotalAssets");
```

---

## 說明
「資產負債表」財務報表項目。凡透過各種交易或其他事項所獲得或控制之經濟資源，能以貨幣衡量並預期未來能提供經濟效益者。
