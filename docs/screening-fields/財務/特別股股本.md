# 特別股股本 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 特別股股本 |
| 欄位分類 | 財務 |
| 代碼 | `CapitalPreferredStocks` |
| 單位 | 億元。 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「資產負債表」股東權益項目中的「特別股股本」。  

```pascal
Value1 = GetField("特別股股本");
Value1 = GetField("CapitalPreferredStocks");
```
