# 每股自由現金流量 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 每股自由現金流量 |
| 欄位分類 | 財務 |
| 代碼 | `FCFFPerShare` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 計算公式：自由現金流量／加權平均股數  

```pascal
Value1 = GetField("每股自由現金流量");
Value1 = GetField("FCFFPerShare");
```
