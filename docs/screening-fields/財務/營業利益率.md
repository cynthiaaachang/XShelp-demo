# 營業利益率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 營業利益率 |
| 欄位分類 | 財務 |
| 代碼 | `OperatingProfitMargin` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 財務比率表上的「營業利益率」欄位。  
> 提供單季與年度資料。  
> 不提供此欄位的公司，輸出值會以零表示。  
> **單位為%，例如：營業利益率為30%，則回傳30。**  

```pascal
Value1 = GetField("營業利益率");
Value1 = GetField("OperatingProfitMargin");
```
