# 資本支出營收比 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 資本支出營收比 |
| 欄位分類 | 財務 |
| 代碼 | `Capitalexpenditure／Sales` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 計算公式 : 現金流量表中 購置不動產廠房設備（含預付）/營業收入淨額*100%。  

```pascal
Value1 = GetField("資本支出營收比");
Value1 = GetField("Capitalexpenditure／Sales");
```

---
