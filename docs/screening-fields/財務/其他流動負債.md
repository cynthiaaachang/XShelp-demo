# 其他流動負債 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 其他流動負債 |
| 欄位分類 | 財務 |
| 代碼 | `OtherCurrentLiabilities` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「資產負債表」中「流動負債」的「其他流動負債」項目。  
> 金融、壽險、產險股不適用此欄位。  

```pascal
Value1 = GetField("其他流動負債");
Value1 = GetField("OtherCurrentLiabilities");
```

---

## 說明
「資產負債表」中「流動負債」的「其他流動負債」項目。
金融、壽險、產險股不適用此欄位。
