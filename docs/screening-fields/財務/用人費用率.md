# 用人費用率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 用人費用率 |
| 欄位分類 | 財務 |
| 代碼 | `Employee Fee ％` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 財務比率表上的「用人費用率」欄位。  
> 提供單季與年度資料。  
> **單位為%，例如：用人費用率為15%，則回傳15。**  

```pascal
Value1 = GetField("用人費用率");
Value1 = GetField("Employee Fee ％");
```
