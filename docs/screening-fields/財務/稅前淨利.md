# 稅前淨利 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 稅前淨利 |
| 欄位分類 | 財務 |
| 代碼 | `Pre－TaxIncome` |
| 單位 | 台股：百萬。美股：百萬(美元) |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 「損益表」中「稅前淨利」項目。  

```pascal
Value1 = GetField("稅前淨利");
Value1 = GetField("Pre－TaxIncome");
```

---

## 說明
「損益表」中「稅前淨利」項目。
凡營業利益（或損失）加上營業外利益（或損失）後之餘額屬之。其為正數，表示本期稅前淨利之數；其為負數，表示本期稅前淨損之數。
