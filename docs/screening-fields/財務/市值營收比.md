# 市值營收比 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 市值營收比 |
| 欄位分類 | 財務 |
| 代碼 | `MarketValue／Sales` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 計算公式: 市值 / 當期營業收入淨額。  

```pascal
Value1 = GetField("市值營收比");
Value1 = GetField("MarketValue／Sales");
```
