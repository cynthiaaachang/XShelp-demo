# 因子分數_實現損益偏態係數 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 因子分數_實現損益偏態係數 |
| 欄位分類 | 財務 |
| 代碼 | `FactorScore_RDSKEW`<br>`因子分數_偏態係數因子(以實現損益為基準)`<br>`FactorScore_Realized skewness Factor` |
