# 10年年化報酬率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 10年年化報酬率 |
| 欄位分類 | 財務 |
| 代碼 | `IRR10YAnnualized` |
| 單位 | % |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股、16.US.E |

## 語法
> 近10年年化報酬率，計算方式包含配息，以及配息再投入。  

```pascal
Value1 = GetField("10年年化報酬率", "D");
Value1 = GetField("IRR10YAnnualized", "D");
```

---

## 說明
近10年年化報酬率，計算方式包含配息，以及配息再投入。

公式是期間的累計報酬率然後年化。
僅支援台/美ETF商品。
