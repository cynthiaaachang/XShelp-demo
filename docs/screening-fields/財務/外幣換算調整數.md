# 外幣換算調整數 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 外幣換算調整數 |
| 欄位分類 | 財務 |
| 代碼 | `AdjustmentForFX`<br>`國外機構報表換算差額` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「資產負債表」股東權益項目中的「國外機構報表換算差額」。  
> 金融、壽險、產險、證券股才適用此欄位。  

```pascal
Value1 = GetField("外幣換算調整數");
Value1 = GetField("AdjustmentForFX");
Value1 = GetField("國外機構報表換算差額");
```

---

## 說明
「資產負債表」股東權益項目中的「國外機構報表換算差額」。
金融、壽險、產險、證券股才適用此欄位。
