# 股價自由現金流量比 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 股價自由現金流量比 |
| 欄位分類 | 財務 |
| 代碼 | `Price／FCF` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、季、年 |
| 支援商品 | 台股 |

## 語法
> 計算公式: 股價 / 每股自由現金流量。  

```pascal
Value1 = GetField("股價自由現金流量比");
Value1 = GetField("Price／FCF");
```

---

## 說明
計算公式: 股價 / 每股自由現金流量。
