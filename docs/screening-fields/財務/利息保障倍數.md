# 利息保障倍數 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 利息保障倍數 |
| 欄位分類 | 財務 |
| 代碼 | `TimesInterestEarne` |
| 單位 | 次 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 財務比率表上的「利息保障倍數」欄位。  
> 提供單季與年度資料。  
> 不提供此欄位的公司，輸出值會以零表示。  

```pascal
Value1 = GetField("利息保障倍數");
Value1 = GetField("TimesInterestEarne");
```
