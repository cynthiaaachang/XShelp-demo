# 本期稅後淨利 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 本期稅後淨利 |
| 欄位分類 | 財務 |
| 代碼 | `NetIncome`<br>`歸屬母公司淨利(損)` |
| 單位 | 台股：百萬。美股：百萬(美元) |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 「損益表」中「歸屬母公司淨利（損）」項目。  

```pascal
Value1 = GetField("本期稅後淨利");
Value1 = GetField("NetIncome");
Value1 = GetField("歸屬母公司淨利(損)");
```

---

## 說明
「損益表」中「歸屬母公司淨利（損）」項目。
凡本期淨利（或淨損）歸屬於母公司業主之當期分攤數屬之。本期淨利（淨損）為繼續營業單位本期淨利（或淨損）加計停業單位損益之餘額屬之。
