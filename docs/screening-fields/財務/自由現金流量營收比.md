# 自由現金流量營收比 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 自由現金流量營收比 |
| 欄位分類 | 財務 |
| 代碼 | `FCF／Sales` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 計算公式: 自由現金流量／營業收入淨額*100%。  
> 提供單季與年度資料。  
> **單位為%，例如：欄位數值為25%，則回傳25。**  

```pascal
Value1 = GetField("自由現金流量營收比");
Value1 = GetField("FCF／Sales");
```

---

## 說明
計算公式: 自由現金流量／營業收入淨額*100%。
其中, 自由現金流量是依現金流量表中的「"來自營運之現金流量" + "購置不動產廠房設備（含預付）－CFI" + "支付利息－CFF"」計算之。正值表示現金流入，負值表示現金流出。
提供單季與年度資料。
單位為%，例如欄位數值為25%，則回傳25。
