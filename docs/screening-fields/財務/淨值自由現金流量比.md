# 淨值自由現金流量比 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 淨值自由現金流量比 |
| 欄位分類 | 財務 |
| 代碼 | `NetValue／FCF` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 計算公式: 股東權益總額 / 自由現金流量*100%。  

```pascal
Value1 = GetField("淨值自由現金流量比");
Value1 = GetField("NetValue／FCF");
```

---

## 說明
計算公式: 股東權益總額 / 自由現金流量*100%。
