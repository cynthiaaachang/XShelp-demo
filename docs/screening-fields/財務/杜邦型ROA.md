# 杜邦型ROA – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 杜邦型ROA |
| 欄位分類 | 財務 |
| 代碼 | `ROA－DuPont?Analysis` |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 計算公式: 總資產週轉率(次) * 稅後淨利率。  

```pascal
Value1 = GetField("杜邦型ROA");
Value1 = GetField("ROA－DuPont?Analysis");
```

---

## 說明
計算公式: 總資產週轉率(次) * 稅後淨利率。
