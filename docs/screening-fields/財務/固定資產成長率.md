# 固定資產成長率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 固定資產成長率 |
| 欄位分類 | 財務 |
| 代碼 | `YOY％－FixedAssets`<br>`折舊性FA成長率` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 固定資產，是資產負債表中非流動資產項下的"不動產廠房及設備"。  
> 固定資產成長率，表示當期"不動產廠房及設備"較去年同期的成長幅度。  
> 提供季度與年度資料。  
> **單位為%，例如：固定資產成長率為25%，則回傳25。**  

```pascal
Value1 = GetField("固定資產成長率");
Value1 = GetField("YOY％－FixedAssets");
Value1 = GetField("折舊性FA成長率");
```

---

## 說明
固定資產，是資產負債表中非流動資產項下的"不動產廠房及設備"。
固定資產成長率，表示當期"不動產廠房及設備"較去年同期的成長幅度。
提供季度與年度資料。
例如：今年第一季"不動產廠房及設備"為100億、去年第一季為80億，固定資產成長率為(100-80)/80*100% = 25%，單位為%，則回傳25。
