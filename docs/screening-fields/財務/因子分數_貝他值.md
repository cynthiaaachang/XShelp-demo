# 因子分數_貝他值 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 因子分數_貝他值 |
| 欄位分類 | 財務 |
| 代碼 | `FactorScore_BETA_1Y`<br>`因子分數_貝他因子`<br>`FactorScore_BETA_1year Factor` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 貝他因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。  

```pascal
Value1 = GetField("因子分數_貝他值", "M");
Value1 = GetField("FactorScore_BETA_1Y", "M");
Value1 = GetField("因子分數_貝他因子", "M");
Value1 = GetField("FactorScore_BETA_1year Factor", "M");
```

---

## 說明
貝他因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。
