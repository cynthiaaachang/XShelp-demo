# 因子_特質波動度 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 因子_特質波動度 |
| 欄位分類 | 財務 |
| 代碼 | `Factor_IVOL`<br>`因子_特質波動度因子`<br>`Factor_Idiosyncratic Factor` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 透過對歷史股票報酬與市場報酬進行Fama French三因子統計迴歸分析，並計算迴歸殘差值的標準差，即為特質波動度因子。  

```pascal
Value1 = GetField("因子_特質波動度", "M");
Value1 = GetField("Factor_IVOL", "M");
Value1 = GetField("因子_特質波動度因子", "M");
Value1 = GetField("Factor_Idiosyncratic Factor", "M");
```

---

## 說明
透過對歷史股票報酬與市場報酬進行Fama French三因子統計迴歸分析，並計算迴歸殘差值的標準差，即為特質波動度因子。
Fama French三因子是透過將市場上的股票分組後統計出的市場溢酬、規模溢酬、以及價值溢酬。
