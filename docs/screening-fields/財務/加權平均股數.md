# 加權平均股數 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 加權平均股數 |
| 欄位分類 | 財務 |
| 代碼 | `WeightedAverageCapitalStock` |
| 單位 | 台股：百萬股。美股：百萬股 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 「損益表」中「加權平均股數」項目。  

```pascal
Value1 = GetField("加權平均股數");
Value1 = GetField("WeightedAverageCapitalStock");
Value1 = GetField("");
```

---

## 說明
「損益表」中「加權平均股數」項目。
加權平均股數是依股數流通期間佔會計期間的比例計算之股數。
