# 利息支出率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 利息支出率 |
| 欄位分類 | 財務 |
| 代碼 | `InterestExpense(％)` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 財務比率表上的「利息支出率」欄位。  
> 提供單季與年度資料。  
> 只支援證券、期貨與金控個股。不提供此欄位的公司，輸出值會以零表示。  
> **單位為%，例如：利息支出率為25%，則回傳25。**  

```pascal
Value1 = GetField("利息支出率");
Value1 = GetField("InterestExpense(％)");
```
