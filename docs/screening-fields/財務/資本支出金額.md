# 資本支出金額 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 資本支出金額 |
| 欄位分類 | 財務 |
| 代碼 | `CapitalExpenditure` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 現金流量表內的"購置不動產廠房設備（含預付）－CFI"項目。現金流量表以負值表示支出。  

```pascal
Value1 = GetField("資本支出金額");
Value1 = GetField("CapitalExpenditure");
```

---
