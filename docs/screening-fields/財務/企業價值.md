# 企業價值 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 企業價值 |
| 欄位分類 | 財務 |
| 代碼 | `EnterpriseValue` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 計算公式: 總市值 ＋ 負債總額 － 現金及約當現金。  

```pascal
Value1 = GetField("企業價值");
Value1 = GetField("EnterpriseValue");
```
