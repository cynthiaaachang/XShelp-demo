# 固定資產報酬率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 固定資產報酬率 |
| 欄位分類 | 財務 |
| 代碼 | `ReturnOnFixedAsset` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 計算公式: 稅後淨利／固定資產*100%。  
> 取單期財報數值來計算。  

```pascal
Value1 = GetField("固定資產報酬率");
Value1 = GetField("ReturnOnFixedAsset");
```
