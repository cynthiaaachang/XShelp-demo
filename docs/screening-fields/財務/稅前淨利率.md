# 稅前淨利率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 稅前淨利率 |
| 欄位分類 | 財務 |
| 代碼 | `Pre－TaxProfitMargin`<br>`Pre－TaxIncomeMargin` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 財務比率表上的「稅前淨利率」欄位。  
> 提供單季與年度資料。  
> **單位為%，例如：稅前淨利率為30%，則回傳30。**  

```pascal
Value1 = GetField("稅前淨利率");
Value1 = GetField("Pre－TaxProfitMargin");
Value1 = GetField("Pre－TaxIncomeMargin");
```

---
