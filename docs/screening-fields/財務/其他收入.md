# 其他收入 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 其他收入 |
| 欄位分類 | 財務 |
| 代碼 | `OtherNonOperatingIncome` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「損益表」中「其他收入－其他」項目。  

```pascal
Value1 = GetField("其他收入");
Value1 = GetField("OtherNonOperatingIncome");
```
