# 因子分數_股票發行量 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 因子分數_股票發行量 |
| 欄位分類 | 財務 |
| 代碼 | `FactorScore_ISSUE`<br>`因子分數_股票發行量因子`<br>`FactorScore_Share issuance Factor` |

