# 營業外收入合計 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 營業外收入合計 |
| 欄位分類 | 財務 |
| 代碼 | `TotalNonOperatingIncome` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「損益表」中「其他收入」項目。  

```pascal
Value1 = GetField("營業外收入合計");
Value1 = GetField("TotalNonOperatingIncome");
```

---

## 說明
「損益表」中「其他收入」項目。凡投資性不動產收入、租賃收入、代理收入及處分投資性不動產利益等屬之。
