# 研發費用 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 研發費用 |
| 欄位分類 | 財務 |
| 代碼 | `OperatingExpense－R＆D` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「損益表」中「研究發展費」項目。  
> 金融、壽險、產險、證券股不適用此欄位。  

```pascal
Value1 = GetField("研發費用");
Value1 = GetField("OperatingExpense－R＆D");
```

---
