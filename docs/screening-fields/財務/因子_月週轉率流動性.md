# 因子_月週轉率流動性 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 因子_月週轉率流動性 |
| 欄位分類 | 財務 |
| 代碼 | `Factor_STDTURN`<br>`因子_流動性因子(以股票月週轉率標準差為基準)`<br>`Factor_Standard deviation of share turnover Factor` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 自然對數(標準差(股票過去36個月的月週轉率))。  

```pascal
Value1 = GetField("因子_月週轉率流動性", "M");
Value1 = GetField("Factor_STDTURN", "M");
Value1 = GetField("因子_流動性因子(以股票月週轉率標準差為基準)", "M");
Value1 = GetField("Factor_Standard deviation of share turnover Factor", "M");
```

## 說明
自然對數(標準差(股票過去36個月的月週轉率))。
月週轉率=月成交量/發行量。
