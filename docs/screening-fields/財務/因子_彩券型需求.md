# 因子_彩券型需求 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 因子_彩券型需求 |
| 欄位分類 | 財務 |
| 代碼 | `Factor_MAX`<br>`因子_彩券型需求因子`<br>`Factor_Lottery Demand Factor` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 過去一個月中最大五筆日報酬率的簡單算數平均。  

```pascal
Value1 = GetField("因子_彩券型需求", "M");
Value1 = GetField("Factor_MAX", "M");
Value1 = GetField("因子_彩券型需求因子", "M");
Value1 = GetField("Factor_Lottery Demand Factor", "M");
```

---

## 說明
過去一個月中最大五筆日報酬率的簡單算數平均。
