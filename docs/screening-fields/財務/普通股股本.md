# 普通股股本 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 普通股股本 |
| 欄位分類 | 財務 |
| 代碼 | `CapitalCommonStocks` |
| 單位 | 台股：億元。美股：百萬(美元) |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 台股:「資產負債表」股東權益項目中的「普通股股本」。  
> 美股:「資產負債表」股東權益項目中的「普通股股東權益」。  

```pascal
Value1 = GetField("普通股股本");
Value1 = GetField("CapitalCommonStocks");
```
