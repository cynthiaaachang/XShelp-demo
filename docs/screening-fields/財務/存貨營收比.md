# 存貨營收比 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 存貨營收比 |
| 欄位分類 | 財務 |
| 代碼 | `Inventory／Sales` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 年 |
| 支援商品 | 台股 |

## 語法
> 計算公式: 存貨之商品及製成品 / 營業收入淨額*100%。  

```pascal
Value1 = GetField("存貨營收比", "Y");
Value1 = GetField("Inventory／Sales", "Y");
```

---

## 說明
計算公式: 存貨之商品及製成品 / 營業收入淨額*100%。
