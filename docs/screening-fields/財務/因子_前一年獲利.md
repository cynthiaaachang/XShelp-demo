# 因子_前一年獲利 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 因子_前一年獲利 |
| 欄位分類 | 財務 |
| 代碼 | `Factor_PROFITABILITY_LAG1`<br>`因子_獲利因子(以前一年獲利為基準)`<br>`Factor_one year lagged profitability Factor` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 一年前的營業利益/三年前的帳面權益。  

```pascal
Value1 = GetField("因子_前一年獲利", "M");
Value1 = GetField("Factor_PROFITABILITY_LAG1", "M");
Value1 = GetField("因子_獲利因子(以前一年獲利為基準)", "M");
Value1 = GetField("Factor_one year lagged profitability Factor", "M");
```

---

## 說明
一年前的營業利益/三年前的帳面權益。
帳面權益=總資產-總負債+遞延所得稅-特別股股本。
