# 利息收入 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 利息收入 |
| 欄位分類 | 財務 |
| 代碼 | `InterestIncome` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「損益表」中「利息收入」項目。  
> 證券股不適用此欄位。  

```pascal
Value1 = GetField("利息收入");
Value1 = GetField("InterestIncome");
```

---

## 說明
「損益表」中「利息收入」項目。凡非以投資為營業項目者，其各種存款、貸出款項及投資金融資產所產生之利息收入屬之。
證券股不適用此欄位。
