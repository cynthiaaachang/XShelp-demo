# 資產報酬率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 資產報酬率 |
| 欄位分類 | 財務 |
| 代碼 | `ROA`<br>`ReturnOnAssets` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 台股，是財務比率表的「ROA(A)稅後息前」項目。  
> 美股，是財務比率表上的「資產報酬率」項目。  
> 提供單季與年度資料。  
> **單位為%，例如：資產報酬率為30%，則回傳30。**  

```pascal
Value1 = GetField("資產報酬率");
Value1 = GetField("ROA");
Value1 = GetField("ReturnOnAssets");
```
