# 營業利益 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 營業利益 |
| 欄位分類 | 財務 |
| 代碼 | `OperatingIncome` |
| 單位 | 台股：百萬。美股：百萬(美元) |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 「損益表」中「營業利益」項目。  

```pascal
Value1 = GetField("營業利益");
Value1 = GetField("OperatingIncome");
```
