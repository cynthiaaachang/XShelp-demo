# 盈餘成長係數 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 盈餘成長係數 |
| 欄位分類 | 財務 |
| 代碼 | `PEG` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 年 |
| 支援商品 | 台股 |

## 語法
> **計算公式: 本益比 / 盈餘成長率 *100%，其中盈餘成長率 = (1-股利合計/EPS) * ROE。**  

```pascal
Value1 = GetField("盈餘成長係數", "Y");
Value1 = GetField("PEG", "Y");
```
