# 兌換損失 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 兌換損失 |
| 欄位分類 | 財務 |
| 代碼 | `ExchangeLoss` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「損益表」中「外幣兌換損失」項目。  

```pascal
Value1 = GetField("兌換損失");
Value1 = GetField("ExchangeLoss");
```

---

## 說明
「損益表」中「外幣兌換損失」項目。凡外幣資產或負債因匯率變動實際兌換、評價之損失屬之。
