# 現金流量比率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 現金流量比率 |
| 欄位分類 | 財務 |
| 代碼 | `CFO ／CL ％` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 財務比率表上的「現金流量比率」欄位。  
> 提供單季與年度資料。  
> **例如：現金流量比率為30%，單位為%，則回傳30。**  
> 不提供此欄位的公司，輸出值會以零表示。  

```pascal
Value1 = GetField("現金流量比率");
Value1 = GetField("CFO ／CL ％");
```
