# 經常利益 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 經常利益 |
| 欄位分類 | 財務 |
| 代碼 | `OrdinaryIncome`<br>`繼續營業單位損益` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「損益表」中「繼續營業單位損益」項目。  

```pascal
Value1 = GetField("經常利益");
Value1 = GetField("OrdinaryIncome");
Value1 = GetField("繼續營業單位損益");
```

---

## 說明
「損益表」中「繼續營業單位損益」項目。
凡本期繼續營業單位稅前淨利（或淨損）扣除所得稅費用（或利益）後之餘額屬之。其為正數，表示繼續營業單位本期稅後淨利之數；其為負數，表示繼續營業單位本期稅後淨損之數。
