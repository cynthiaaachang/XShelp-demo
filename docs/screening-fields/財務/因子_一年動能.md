# 因子_一年動能 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 因子_一年動能 |
| 欄位分類 | 財務 |
| 代碼 | `Factor_MOM`<br>`因子_動能因子(以前11個月報酬為基準)`<br>`Factor_Momentum Factor` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 股票在過去1年內(不包含當月，共計11個月)的漲跌幅度。  

```pascal
Value1 = GetField("因子_一年動能", "M");
Value1 = GetField("Factor_MOM", "M");
Value1 = GetField("因子_動能因子(以前11個月報酬為基準)", "M");
Value1 = GetField("Factor_Momentum Factor", "M");
```

---

## 說明
股票在過去1年內(不包含當月，共計11個月)的漲跌幅度。
