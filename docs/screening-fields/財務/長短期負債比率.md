# 長短期負債比率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 長短期負債比率 |
| 欄位分類 | 財務 |
| 代碼 | `LongTermDebt/ShortTermDebt` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 計算公式: 流動負債／負債總額 *100%。  

```pascal
Value1 = GetField("長短期負債比率");
Value1 = GetField("LongTermDebt/ShortTermDebt");
```
