# 因子分數_現金殖利率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 因子分數_現金殖利率 |
| 欄位分類 | 財務 |
| 代碼 | `FactorScore_DY`<br>`因子分數_現金殖利率因子`<br>`FactorScore_Dividend Yield` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 現金殖利率因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。  

```pascal
Value1 = GetField("因子分數_現金殖利率", "M");
Value1 = GetField("FactorScore_DY", "M");
Value1 = GetField("因子分數_現金殖利率因子", "M");
Value1 = GetField("FactorScore_Dividend Yield", "M");
```

---

## 說明
現金殖利率因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。
