# 有息負債利率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 有息負債利率 |
| 欄位分類 | 財務 |
| 代碼 | `Int.Exp.／Debt` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 財務比率表上的「有息負債利率」欄位。  
> 提供單季與年度資料。  
> **單位為%，例如：有息負債利率為1.5%，則回傳1.5。**  
> 不提供此欄位的公司，輸出值會以零表示。  

```pascal
Value1 = GetField("有息負債利率");
Value1 = GetField("Int.Exp.／Debt");
```
