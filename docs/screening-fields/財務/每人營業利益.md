# 每人營業利益 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 每人營業利益 |
| 欄位分類 | 財務 |
| 代碼 | `OperatingIncomePerEmployee` |
| 單位 | 千元 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 財務比率表上的「每人營業利益」欄位。  
> 提供單季與年度資料。  
> 不提供此欄位的公司，輸出值會以零表示。  

```pascal
Value1 = GetField("每人營業利益");
Value1 = GetField("OperatingIncomePerEmployee");
```
