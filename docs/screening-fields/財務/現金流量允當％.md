# 現金流量允當％ – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 現金流量允當％ |
| 欄位分類 | 財務 |
| 代碼 | `CFAdequacyRatio` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 年 |
| 支援商品 | 台股 |

## 語法
> 計算公式 : 最近5年度來自營運之現金流量/ 最近5年度(長期投資(新增)+固定資產(購置)+存貨(增)減+支付現金股利) *100%。  
> 只提供年度頻率資料。  
> **單位為%，例如：欄位數值105%，則回傳105。**  

```pascal
Value1 = GetField("現金流量允當％", "Y");
Value1 = GetField("CFAdequacyRatio", "Y");
```
