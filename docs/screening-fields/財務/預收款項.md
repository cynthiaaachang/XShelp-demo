# 預收款項 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 預收款項 |
| 欄位分類 | 財務 |
| 代碼 | `AdvancesCustomers` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「資產負債表」中「預收款項」項目。  
> 金融股不適用此欄位。  

```pascal
Value1 = GetField("預收款項");
Value1 = GetField("AdvancesCustomers");
```

---

## 說明
「資產負債表」中「預收款項」項目。
金融股不適用此欄位。
