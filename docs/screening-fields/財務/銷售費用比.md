# 銷售費用比 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 銷售費用比 |
| 欄位分類 | 財務 |
| 代碼 | `SellingExpanseRatio／CostOfGoodsSold` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 計算公式: 銷售費用/營業成本  
> 提供單季與年度資料。  
> **單位為%，例如：欄位數值5%，則回傳5。不提供此欄位的公司，輸出值會以零表示。**  

```pascal
Value1 = GetField("銷售費用比");
Value1 = GetField("SellingExpanseRatio／CostOfGoodsSold");
```
