# 應付帳款及票據 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 應付帳款及票據 |
| 欄位分類 | 財務 |
| 代碼 | `AccountsPayable＆NotesPayable` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「資產負債表」中「應付帳款及票據」項目。  
> 金融股不適用此欄位。  

```pascal
Value1 = GetField("應付帳款及票據");
Value1 = GetField("AccountsPayable＆NotesPayable");
```

---

## 說明
「資產負債表」中「應付帳款及票據」項目。
金融股不適用此欄位。
