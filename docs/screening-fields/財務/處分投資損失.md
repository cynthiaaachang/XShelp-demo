# 處分投資損失 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 處分投資損失 |
| 欄位分類 | 財務 |
| 代碼 | `Loss－DisposalInvestment` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「損益表」中「處分投資損失」項目。  

```pascal
Value1 = GetField("處分投資損失");
Value1 = GetField("Loss－DisposalInvestment");
```

---

## 說明
「損益表」中「處分投資損失」項目。凡非以投資為營業項目者，處分金融資產（透過損益按公允價值衡量之金融資產除外）產生之損失等屬之。
金融、壽險、產險、證券股不適用此欄位。
