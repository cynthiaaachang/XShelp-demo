# 應付商業本票 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 應付商業本票 |
| 欄位分類 | 財務 |
| 代碼 | `BillsIssued` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「資產負債表」中「流動負債」項內「應付商業本票∕承兌匯票」項目。  
> 金融、壽險、產險股不適用此欄位。  

```pascal
Value1 = GetField("應付商業本票");
Value1 = GetField("BillsIssued");
```

---

## 說明
「資產負債表」中「流動負債」項內「應付商業本票∕承兌匯票」項目。
金融、壽險、產險股不適用此欄位。
