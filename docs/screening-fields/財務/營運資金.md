# 營運資金 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 營運資金 |
| 欄位分類 | 財務 |
| 代碼 | `WorkingCapital` |
| 單位 | 台股:百萬；美股:百萬(美元) |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 計算公式  : 流動資產 - 流動負債  

```pascal
Value1 = GetField("營運資金");
Value1 = GetField("WorkingCapital");
```
