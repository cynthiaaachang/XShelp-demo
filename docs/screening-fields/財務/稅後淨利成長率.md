# 稅後淨利成長率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 稅後淨利成長率 |
| 欄位分類 | 財務 |
| 代碼 | `YOY％－NetIncome` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 財務比率表上的「稅後淨利成長率」欄位。表示當期稅後淨利較去年同期稅後淨利的成長幅度。  
> 提供單季與年度資料。  
> **單位為%，例如：稅後淨利成長率為25%，則回傳25。**  

```pascal
Value1 = GetField("稅後淨利成長率");
Value1 = GetField("YOY％－NetIncome");
```

---

## 說明
財務比率表上的「稅後淨利成長率」欄位。表示當期稅後淨利較去年同期稅後淨利的成長幅度。
提供單季與年度資料。
例如：今年第一季稅後淨利100億、去年第一季稅後淨利80億，稅後淨利成長率為(100-80)/80*100% = 25%，單位為%，則回傳25。
