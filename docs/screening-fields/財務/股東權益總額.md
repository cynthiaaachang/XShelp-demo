# 股東權益總額 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 股東權益總額 |
| 欄位分類 | 財務 |
| 代碼 | `TotalEquity` |
| 單位 | 台股：百萬。美股：百萬(美元) |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 「資產負債表」財務報表項目。  

```pascal
Value1 = GetField("股東權益總額");
Value1 = GetField("TotalEquity");
```

---

## 說明
「資產負債表」財務報表項目。凡全部資產減除全部負債後之餘額者屬之，包括資本、資本公積、保留盈餘（累積虧損）、權益調整及非控制權益等。
