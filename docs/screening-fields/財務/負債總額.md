# 負債總額 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 負債總額 |
| 欄位分類 | 財務 |
| 代碼 | `TotalLiabilities` |
| 單位 | 台股：百萬。美股：百萬(美元) |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 「資產負債表」財務報表項目。  

```pascal
Value1 = GetField("負債總額");
Value1 = GetField("TotalLiabilities");
```

---

## 說明
「資產負債表」財務報表項目。凡過去交易或其他事項所發生之經濟義務，能以貨幣衡量，並將以提供勞務或支付經濟資源之方式償付者，包括流動負債、存款、匯款及金融債券、央行及同業融資、長期負債、其他負債等。
