# 長期投資 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 長期投資 |
| 欄位分類 | 財務 |
| 代碼 | `Long－TermInvestment` |
| 單位 | 台股:百萬；美股:百萬(美元) |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 資產負債表的4項資料加總  
> 1.透過損益按公允價值衡量之金融資產－非流動  
> 2.透過其他綜合損益按公允價值衡量之金融資產－非流動  
> 3.按攤銷後成本衡量之金融資產－非流動  
> 4.採權益法之長期股權投資  

```pascal
Value1 = GetField("長期投資");
Value1 = GetField("Long－TermInvestment");
```

---

## 說明
資產負債表的4項資料加總 
1.透過損益按公允價值衡量之金融資產－非流動 
2.透過其他綜合損益按公允價值衡量之金融資產－非流動 
3.按攤銷後成本衡量之金融資產－非流動 
4.採權益法之長期股權投資
