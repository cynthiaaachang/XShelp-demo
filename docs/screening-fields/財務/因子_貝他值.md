# 因子_貝他值 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 因子_貝他值 |
| 欄位分類 | 財務 |
| 代碼 | `Factor_BETA_1Y`<br>`因子_貝他因子`<br>`Factor_BETA_1year Factor` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 過去一年的日報酬率和市場報酬率之共變異數/市場報酬率之變異數。  

```pascal
Value1 = GetField("因子_貝他值", "M");
Value1 = GetField("Factor_BETA_1Y", "M");
Value1 = GetField("因子_貝他因子", "M");
Value1 = GetField("Factor_BETA_1year Factor", "M");
```

---

## 說明
過去一年的日報酬率和市場報酬率之共變異數/市場報酬率之變異數。
股票所屬市場視其交易市場而定，上市公司=加權指數報酬率；上櫃公司=櫃買指數報酬率。此因子與「貝他值」不同，是由中興大學財金所提供之因子計算規則計算而得。
