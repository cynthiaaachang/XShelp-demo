# 總流通在外股數 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 總流通在外股數 |
| 欄位分類 | 財務 |
| 代碼 | `TotalSharesOutstanding` |
| 單位 | 股 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 美(股票) |

## 語法
> 「資產負債表」中「流通在外股數」項目。  
> 美股股票才適用本欄位。  

```pascal
Value1 = GetField("總流通在外股數");
Value1 = GetField("TotalSharesOutstanding");
```

## 說明
「資產負債表」中「流通在外股數」項目。
