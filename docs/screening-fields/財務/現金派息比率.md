# 現金派息比率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 現金派息比率 |
| 欄位分類 | 財務 |
| 代碼 | `CurrentPayoutRatio` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 年 |
| 支援商品 | 台股 |

## 語法
> 計算公式: 現金股利 / 每股盈餘*100%。  

```pascal
Value1 = GetField("現金派息比率", "Y");
Value1 = GetField("CurrentPayoutRatio", "Y");
```
