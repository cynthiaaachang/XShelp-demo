# 每股營業利益(元) – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 每股營業利益(元) |
| 欄位分類 | 財務 |
| 代碼 | `OperatingIncomePerShare` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |

## 語法
> 財務比率表上的「每股營業利益(元)」欄位。  
> 提供單季與年度資料。  

```pascal
Value1 = GetField("每股營業利益(元)");
Value1 = GetField("OperatingIncomePerShare");
```
