# 固定資產 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 固定資產 |
| 欄位分類 | 財務 |
| 代碼 | `TotalFixedAssets`<br>`不動產廠房及設備` |
| 單位 | 台股：百萬。美股：百萬(美元) |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 台股:「資產負債表」中的「不動產及設備淨額」項目。  
> 美股: 「資產負債表」中的「財產，機器及設備淨額」項目。  

```pascal
Value1 = GetField("固定資產");
Value1 = GetField("TotalFixedAssets");
Value1 = GetField("不動產廠房及設備");
```

---

## 說明
台股:「資產負債表」中的「不動產及設備淨額」項目。
美股: 「資產負債表」中的「財產，機器及設備淨額」項目。
