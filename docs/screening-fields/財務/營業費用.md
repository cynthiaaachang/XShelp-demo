# 營業費用 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 營業費用 |
| 欄位分類 | 財務 |
| 代碼 | `OperatingExpenses` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「損益表」中「營業費用」項目。  
> 證券股不適用此欄位。  

```pascal
Value1 = GetField("營業費用");
Value1 = GetField("OperatingExpenses");
```

---

## 說明
「損益表」中「營業費用」項目。凡配合產銷營運所發生之各項營業費用皆屬之。
證券股不適用此欄位。
