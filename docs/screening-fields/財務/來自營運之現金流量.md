# 來自營運之現金流量 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 來自營運之現金流量 |
| 欄位分類 | 財務 |
| 代碼 | `CashFlow－Operating`<br>`CFO` |
| 單位 | 台股：百萬。美股：百萬(美元) |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 現金流量表上的「來自營運之現金流量」欄位。  
> 正值表示現金流入，負值表示現金流出。  
> 提供單季與年度資料。  

```pascal
Value1 = GetField("來自營運之現金流量");
Value1 = GetField("CashFlow－Operating");
Value1 = GetField("CFO");
```
