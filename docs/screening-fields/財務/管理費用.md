# 管理費用 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 管理費用 |
| 欄位分類 | 財務 |
| 代碼 | `OperatingExpense－ADM` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「損益表」中「管理費用」項目。  
> 金融、證券股不適用此欄位。  

```pascal
Value1 = GetField("管理費用");
Value1 = GetField("OperatingExpense－ADM");
```

---

## 說明
「損益表」中「管理費用」項目。凡各項管理費用屬之。
金融、證券股不適用此欄位。
