# 因子分數_成交值流動性 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 因子分數_成交值流動性 |
| 欄位分類 | 財務 |
| 代碼 | `FactorScore_DVOL`<br>`因子分數_流動性因子(以成交值為基準)`<br>`FactorScore_Trading volume Factor` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 流動性因子(以成交值為基準)經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。  

```pascal
Value1 = GetField("因子分數_成交值流動性", "M");
Value1 = GetField("FactorScore_DVOL", "M");
Value1 = GetField("因子分數_流動性因子(以成交值為基準)", "M");
Value1 = GetField("FactorScore_Trading volume Factor", "M");
```
