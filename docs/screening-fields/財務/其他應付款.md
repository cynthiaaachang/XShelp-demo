# 其他應付款 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 其他應付款 |
| 欄位分類 | 財務 |
| 代碼 | `OtherReceivable` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「資產負債表」中負債項的「其他應付款」或「應付款項」項目。  

```pascal
Value1 = GetField("其他應付款");
Value1 = GetField("OtherReceivable");
```
