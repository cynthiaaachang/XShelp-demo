# 短期借支 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 短期借支 |
| 欄位分類 | 財務 |
| 代碼 | `LoanToOther－ShortTerm` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「資產負債表」中「流動資產」項內的「資金貸予他人－流動」項目。  
> 金融、壽險、產險、證券股不適用此欄位。  

```pascal
Value1 = GetField("短期借支");
Value1 = GetField("LoanToOther－ShortTerm");
```

---

## 說明
「資產負債表」中「流動資產」項內的「資金貸予他人－流動」項目。
金融、壽險、產險、證券股不適用此欄位。
