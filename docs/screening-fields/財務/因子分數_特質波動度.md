# 因子分數_特質波動度 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 因子分數_特質波動度 |
| 欄位分類 | 財務 |
| 代碼 | `FactorScore_IVOL`<br>`因子分數_特質波動度因子`<br>`FactorScore_Idiosyncratic Factor` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 特質波動度因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。  

```pascal
Value1 = GetField("因子分數_特質波動度", "M");
Value1 = GetField("FactorScore_IVOL", "M");
Value1 = GetField("因子分數_特質波動度因子", "M");
Value1 = GetField("FactorScore_Idiosyncratic Factor", "M");
```

---

## 說明
特質波動度因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。
