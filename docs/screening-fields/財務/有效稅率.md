# 有效稅率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 有效稅率 |
| 欄位分類 | 財務 |
| 代碼 | `Tax Rate` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 財務比率表上的「稅率(A)」欄位。  
> 提供單季與年度資料。  
> 若稅前淨利或所得稅費用為負或0、所得稅費用大於稅前淨利等特殊情況，資料會呈現為0。  

```pascal
Value1 = GetField("有效稅率");
Value1 = GetField("Tax Rate");
```

---

## 說明
財務比率表上的「稅率(A)」欄位。
提供單季與年度資料。
若稅前淨利或所得稅費用為負或0、所得稅費用大於稅前淨利等特殊情況，資料會呈現為0。
