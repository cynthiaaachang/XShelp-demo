# 因子_潛在上檔報酬比 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 因子_潛在上檔報酬比 |
| 欄位分類 | 財務 |
| 代碼 | `Factor_UPR`<br>`因子_潛在上檔報酬比率因子`<br>`Factor_Upside Potential Ratio Factor` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 將過去1年的日報酬率隨機抽樣1000筆，以正報酬率的加總與過去1年日交易天數的比值作為分子，以負報酬率的平方加總與過去1年日交易天數的比值平方根作為分母，得到的比率即為潛在上檔報酬比率因子。  

```pascal
Value1 = GetField("因子_潛在上檔報酬比", "M");
Value1 = GetField("Factor_UPR", "M");
Value1 = GetField("因子_潛在上檔報酬比率因子", "M");
Value1 = GetField("Factor_Upside Potential Ratio Factor", "M");
```

---
