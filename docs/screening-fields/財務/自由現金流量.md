# 自由現金流量 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 自由現金流量 |
| 欄位分類 | 財務 |
| 代碼 | `FCFF` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 現金流量表中的「"來自營運之現金流量" + "購置不動產廠房設備（含預付）－CFI" + "支付利息－CFF"」。  
> 正值表示現金流入，負值表示現金流出。  

```pascal
Value1 = GetField("自由現金流量");
Value1 = GetField("FCFF");
```
