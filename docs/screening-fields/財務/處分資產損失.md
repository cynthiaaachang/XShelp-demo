# 處分資產損失 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 處分資產損失 |
| 欄位分類 | 財務 |
| 代碼 | `Loss－DisposalFixedAssets` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「損益表」中「處分不動產、廠房及設備損失」項目。  

```pascal
Value1 = GetField("處分資產損失");
Value1 = GetField("Loss－DisposalFixedAssets");
```

---

## 說明
「損益表」中「處分不動產、廠房及設備損失」項目。凡不動產、廠房及設備變賣、交換等所產生之損失屬之。
