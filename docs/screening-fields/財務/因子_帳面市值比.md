# 因子_帳面市值比 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 因子_帳面市值比 |
| 欄位分類 | 財務 |
| 代碼 | `Factor_BM`<br>`因子_帳面市值比因子`<br>`Factor_Book-to-Market ratio Factor` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 自然對數(帳面價值/市場價值)。  

```pascal
Value1 = GetField("因子_帳面市值比", "M");
Value1 = GetField("Factor_BM", "M");
Value1 = GetField("因子_帳面市值比因子", "M");
Value1 = GetField("Factor_Book-to-Market ratio Factor", "M");
```

---

## 說明
自然對數(帳面價值/市場價值)。
帳面價值=股東權益總額+遞延所得稅。
市場價值=股票總市值。
