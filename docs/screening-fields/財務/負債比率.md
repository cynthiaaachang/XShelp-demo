# 負債比率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 負債比率 |
| 欄位分類 | 財務 |
| 代碼 | `LiabilitiesRatio` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 財務比率表上的「負債比率」欄位。  
> 提供季度與年度資料。  
> **單位為%，例如：負債比率為30%，則回傳30。**  

```pascal
Value1 = GetField("負債比率");
Value1 = GetField("LiabilitiesRatio");
```
