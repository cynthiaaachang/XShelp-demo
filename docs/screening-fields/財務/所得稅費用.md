# 所得稅費用 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 所得稅費用 |
| 欄位分類 | 財務 |
| 代碼 | `IncomeTaxExpense` |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 「損益表」中「所得稅費用」項目。  

```pascal
Value1 = GetField("所得稅費用");
Value1 = GetField("IncomeTaxExpense");
```

---

## 說明
「損益表」中「所得稅費用」項目。
凡本期稅前淨利（或淨損），依所得稅法等有關規定核算應認列之所得稅費用（或利益）屬之。
