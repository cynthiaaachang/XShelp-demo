# 淨值成長率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 淨值成長率 |
| 欄位分類 | 財務 |
| 代碼 | `YOY％－TotalEquity` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股 |

## 語法
> 財務比率表上的「淨值成長率」欄位。表示當期股東權益總額較去年同期股東權益總額的成長幅度。  
> 提供季度與年度資料。  
> **單位為%，例如：淨值成長率為25%，則回傳25。**  

```pascal
Value1 = GetField("淨值成長率");
Value1 = GetField("YOY％－TotalEquity");
```

---

## 說明
財務比率表上的「淨值成長率」欄位。表示當期股東權益總額較去年同期股東權益總額的成長幅度。
股東權益＝資產總額－負債總額，也稱為淨值或凈資產。
淨值成長率，提供季度與年度資料。
例如：今年第一季股東權益總額100億、去年第一季股東權益總額80億，淨值成長率為(100-80)/80*100% = 25%，單位為%，則回傳25。
