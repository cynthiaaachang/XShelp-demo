# 外銷比率 – （選股欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 外銷比率 |
| 欄位分類 | 財務 |
| 代碼 | `ExportRatio` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 年 |
| 支援商品 | 台股 |

## 語法
> 外銷占營收的比例。  
> **單位為％，例如：外銷比率為90%，則回傳90。**  

```pascal
Value1 = GetField("外銷比率", "Y");
Value1 = GetField("ExportRatio", "Y");
```

---

## 說明
外銷占營收的比例。
例如：公司年度總營收為100億，外銷地區的總營收為80億，外銷比率為80%，單位為％，則回傳80。
