# 股票股利 – （選股欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 股票股利 |
| 欄位分類 | 基本 |
| 代碼 | `StockDividend` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 年 |
| 支援商品 | 台股 |

## 語法
> 最近一個完整股利年度的股票股利。  

```pascal
Value1 = GetField("股票股利", "Y");
Value1 = GetField("StockDividend", "Y");
```

---

## 說明
最近一個完整股利年度的股票股利。
