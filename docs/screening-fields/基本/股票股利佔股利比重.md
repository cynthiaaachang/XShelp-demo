# 股票股利佔股利比重 – （選股欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 股票股利佔股利比重 |
| 欄位分類 | 基本 |
| 代碼 | `StockDividend／TotalDividend` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 年 |
| 支援商品 | 台股 |

## 語法
> 計算公式：股票股利 / (現金股利+股票股利) *100%。  

```pascal
Value1 = GetField("股票股利佔股利比重", "Y");
Value1 = GetField("StockDividend／TotalDividend", "Y");
```

---

## 說明
計算公式：股票股利 / (現金股利+股票股利) *100%。
