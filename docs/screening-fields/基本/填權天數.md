# 填權天數 – （選股欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 填權天數 |
| 欄位分類 | 基本 |
| 代碼 | `DaysForStockDividendAdjustment` |
| 單位 | 天 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 語法
> **填權天數。單位是天，例如填權天數為1天的話，則回傳1。若尚未除權的話，則回傳-2。若尚未填權的話，則回傳-1。**  

```pascal
Value1 = GetField("填權天數");
Value1 = GetField("DaysForStockDividendAdjustment");
```

---

## 說明
填權天數，指的是在股票除權後，股價從除權參考價回升至除權前收盤價所需的天數。

單位是天，例如填權天數為1天的話，則回傳1。若尚未除權的話，則回傳-2。若尚未填權的話，則回傳-1。
