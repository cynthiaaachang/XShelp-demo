# 公積配股 – （選股欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 公積配股 |
| 欄位分類 | 基本 |
| 代碼 | `DividendFromCapitalReserve` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 年 |
| 支援商品 | 台股 |

## 語法
> 股票股利內的公積配股。  
> **(股票股利 = 盈餘配股 + 公積配股)**  

```pascal
Value1 = GetField("公積配股", "Y");
Value1 = GetField("DividendFromCapitalReserve", "Y");
```

---

## 說明
股票股利內的公積配股。
(股票股利 = 盈餘配股 + 公積配股)
