# 員工人數 – （選股欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 員工人數 |
| 欄位分類 | 基本 |
| 代碼 | `Numemployees` |
| 單位 | 人數 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季、年 |
| 支援商品 | 台股、美(股票) |

## 語法
> 來自年報的員工總人數。  
> 註：美股僅支援年頻率。  

```pascal
Value1 = GetField("員工人數");
Value1 = GetField("Numemployees");
```
