# 股本(億) – （選股欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 股本(億) |
| 欄位分類 | 基本 |
| 代碼 | `Capital-Lastest`<br>`最新股本` |
| 單位 | 億元 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股 |

## 語法
> 股票的股本，換算成億元。  

```pascal
Value1 = GetField("股本(億)", "D");
Value1 = GetField("Capital-Lastest", "D");
Value1 = GetField("最新股本", "D");
```

---

## 說明
股票的股本，換算成億元。
