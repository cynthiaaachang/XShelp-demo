# 公司類別 – （選股欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 公司類別 |
| 欄位分類 | 基本 |
| 代碼 | `CompanySize` |
| 格式 | 字串 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 系統運用市值，將公司分為大型股、中型股、小型股。  
> 公司類別數值會每月重新評估。  

```pascal
Value1 = GetField("公司類別", "M");
Value1 = GetField("CompanySize", "M");
```

---
