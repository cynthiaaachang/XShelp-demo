# 累計營收 – （選股欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 累計營收 |
| 欄位分類 | 基本 |
| 代碼 | `AccSales` |
| 單位 | 億 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 年初迄今公告的月營收累計總金額。  

```pascal
Value1 = GetField("累計營收", "M");
Value1 = GetField("AccSales", "M");
```

---

## 說明
年初迄今公告的月營收累計總金額。

例如, 一月月營收5億, 二月月營收, 5億, 三月月營收5億，三月的累計營收為15億。
