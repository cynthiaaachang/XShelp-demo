# 現金股利 – （選股欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 現金股利 |
| 欄位分類 | 基本 |
| 代碼 | `CashDividend`<br>`股息` |
| 單位 | 台股:元；美股:元(美元) |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 年 |
| 支援商品 | 台股、美(股票)、16.US.P、16.US.E |

## 語法
> 現金配發的股利(股息)。  

```pascal
Value1 = GetField("現金股利", "Y");
Value1 = GetField("CashDividend", "Y");
Value1 = GetField("股息", "Y");
```

---

## 說明
現金配發的股利(股息)。

統計方式依照商品類型有些許差異：

台股: 回傳最近一個完整的股利年度所配發的所有現金股利。
美股: 最近12個月(1年)的現金股息的加總。
ETF: 最近12個月(1年)的現金股息的加總。
