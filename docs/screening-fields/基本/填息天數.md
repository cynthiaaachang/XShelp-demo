# 填息天數 – （選股欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 填息天數 |
| 欄位分類 | 基本 |
| 代碼 | `DaysForCashDividendAdjustment` |
| 單位 | 天 |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 語法
> **填息天數。單位是天，例如填息天數為1天的話，則回傳1。若尚未除息的話，則回傳-2。若尚未填息的話，則回傳-1。**  

```pascal
Value1 = GetField("填息天數");
Value1 = GetField("DaysForCashDividendAdjustment");
```

---

## 說明
填息天數，指的是在股票除息後，股價從除息參考價回升至除息前收盤價所需的天數。

單位是天，例如填息天數為1天的話，則回傳1。若尚未除息的話，則回傳-2。若尚未填息的話，則回傳-1。
