# 現金股利殖利率 – （選股欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 現金股利殖利率 |
| 欄位分類 | 基本 |
| 代碼 | `CashDividendYield` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股 |

## 語法
> 現金股利相對於股價的比例，用來衡量股票投資的回報。  

```pascal
Value1 = GetField("現金股利殖利率", "D");
Value1 = GetField("CashDividendYield", "D");
```

---

## 說明
現金股利相對於股價的比例，用來衡量股票投資的回報。

計算公式依股票或是ETF有些許不同：

台股股票：最近一個完整股利年度的現金股利／收盤價 * 100%

ETF：最近12個月股利／收盤價 * 100%。
