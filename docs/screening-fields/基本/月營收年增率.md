# 月營收年增率 – （選股欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 月營收年增率 |
| 欄位分類 | 基本 |
| 代碼 | `Sales－YoY` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 商品當月營收較去年同月營收的成長率。  
> 計算公式 ＝(當月營收－去年同期) / 去年同期*100%。  
> **單位為%，例如月營收年增率為10%，則回傳10。**  

```pascal
Value1 = GetField("月營收年增率", "M");
Value1 = GetField("Sales－YoY", "M");
```

---

## 說明
商品當月營收較去年同月營收的成長率。
計算公式 ＝(當月營收－去年同期) / 去年同期*100%。

例如：今年三月營收100億、去年三月營收80億，月營收年增率為(100-80)/80*100% = 25%，單位為%， 則回傳25。
