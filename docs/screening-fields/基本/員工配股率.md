# 員工配股率 – （選股欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 員工配股率 |
| 欄位分類 | 基本 |
| 代碼 | `RatioOfEmployeeBonus－Stock` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 年 |
| 支援商品 | 台股 |

## 語法
> 計算公式  : 員工分紅股數 / 已發行股數*100%。  
> 資料來源：台灣經濟新報。  

```pascal
Value1 = GetField("員工配股率", "Y");
Value1 = GetField("RatioOfEmployeeBonus－Stock", "Y");
```
