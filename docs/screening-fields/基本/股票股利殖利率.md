# 股票股利殖利率 – （選股欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 股票股利殖利率 |
| 欄位分類 | 基本 |
| 代碼 | `StockDividendYield` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股 |

## 語法
> 股票股利(配股)相對於股價的比例。  

```pascal
Value1 = GetField("股票股利殖利率", "D");
Value1 = GetField("StockDividendYield", "D");
```

---

## 說明
股票股利(配股)相對於股價的比例。

計算公式：(最近一年股票股利／收盤價) * 100%。
