# 股本(元) – （選股欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 股本(元) |
| 欄位分類 | 基本 |
| 代碼 | `CurrentCapital` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股 |

## 語法
> 股票的股本，換算成元。  

```pascal
Value1 = GetField("股本(元)", "D");
Value1 = GetField("CurrentCapital", "D");
```
