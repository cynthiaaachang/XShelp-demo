# 公司風格 – （選股欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 公司風格 |
| 欄位分類 | 基本 |
| 代碼 | `CompanyStyle` |
| 格式 | 字串 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 季 |
| 支援商品 | 台股 |

## 語法
> 系統運用財務數值評價，將公司分為混合型、價值型、成長型。  
> 公司風格數值會每季重新評估。  

```pascal
Value1 = GetField("公司風格", "Q");
Value1 = GetField("CompanyStyle", "Q");
```

---

## 說明
系統運用財務數值評價，將公司分為混合型、價值型、成長型。
公司風格數值會每季重新評估。
