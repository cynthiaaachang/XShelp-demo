# 公司掛牌日期 – （選股欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 公司掛牌日期 |
| 欄位分類 | 基本 |
| 代碼 | `ListingDate` |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 語法
> 公司上櫃或是上市的日期。  
> 欄位格式為西元年月日，例如20100101。  

```pascal
Value1 = GetField("公司掛牌日期");
Value1 = GetField("ListingDate");
```
