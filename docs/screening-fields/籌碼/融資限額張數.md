# 融資限額張數 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 融資限額張數 |
| 欄位分類 | 籌碼 |
| 代碼 | `MarginLimit` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 台股 |

## 語法
> 交易所公告的融資限額張數。  

```pascal
Value1 = GetField("融資限額張數");
Value1 = GetField("MarginLimit");
```

---

## 說明
交易所公告的融資限額張數。
