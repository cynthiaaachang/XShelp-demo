# 自營商避險買賣超 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 自營商避險買賣超 |
| 欄位分類 | 籌碼 |
| 代碼 | `DHedgedifference` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 台股 |

## 語法
> 自營商買賣超內歸屬於避險的數量。數值內容/單位依商品類型而異。  

```pascal
Value1 = GetField("自營商避險買賣超");
Value1 = GetField("DHedgedifference");
```

---

## 說明
自營商買賣超內歸屬於避險的數量。數值內容/單位依商品類型而異。

台股商品的數值為自營商避險買張 ─ 自營商避險賣張。
大盤商品的數值則是自營商避險買進金額 ─ 自營商避險賣出金額，單位是元。

週頻率以上為期間累計值。
