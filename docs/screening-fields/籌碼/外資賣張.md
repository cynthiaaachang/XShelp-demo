# 外資賣張 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 外資賣張 |
| 欄位分類 | 籌碼 |
| 代碼 | `Ftotalsell` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 台股 |

## 語法
> 外資賣出張數。  

```pascal
Value1 = GetField("外資賣張");
Value1 = GetField("Ftotalsell");
```

---

## 說明
外資賣出張數。

週頻率以上為期間累計值。
