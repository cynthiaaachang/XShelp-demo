# 融券使用率 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 融券使用率 |
| 欄位分類 | 籌碼 |
| 代碼 | `ShortSell％` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 台股 |

## 語法
> 融券使用的比例。計算公式：融券餘額 / 融資限額 * 100%。  
> **單位是％，例如融券使用率為70%的話，則回傳70。**  

```pascal
Value1 = GetField("融券使用率");
Value1 = GetField("ShortSell％");
```

---

## 說明
融券使用的比例。計算公式：融券餘額 / 融資限額 * 100%。

單位是％，例如融券使用率為70%的話，則回傳70。

週頻率以上為統計期間的期底值。
