# 融券買進張數 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 融券買進張數 |
| 欄位分類 | 籌碼 |
| 代碼 | `ShortSaleRecall` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 台股 |

## 語法
> 融券買進的張數。  

```pascal
Value1 = GetField("融券買進張數");
Value1 = GetField("ShortSaleRecall");
```

---

## 說明
融券買進的張數。

週頻率以上為期間累計值。
