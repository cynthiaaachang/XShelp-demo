# 內部人持股張數 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 內部人持股張數 |
| 欄位分類 | 籌碼 |
| 代碼 | `Insidersharesheld` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> **內部人= 台灣經濟新報所提供的所有董監持股資料 – 大股東。不重複計算股數 (即剃除重複名單/法人名稱資料)。每月更新。依查詢頻率取期底值。**  

```pascal
Value1 = GetField("內部人持股張數", "M");
Value1 = GetField("Insidersharesheld", "M");
```
