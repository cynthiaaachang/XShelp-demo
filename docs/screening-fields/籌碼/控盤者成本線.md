# 控盤者成本線 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 控盤者成本線 |
| 欄位分類 | 籌碼 |
| 代碼 | `CostLine` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月、季、半年、年 |
| 支援商品 | 台股 |

## 語法
> 系統估計的控盤者的持股成本。  

```pascal
Value1 = GetField("控盤者成本線");
Value1 = GetField("CostLine");
```

---

## 說明
系統估計的控盤者的持股成本。

控盤者的資料是每日單一價格成交值大於100萬的買賣記錄。

系統依此資料，估計出控盤者的長期持有成本。
