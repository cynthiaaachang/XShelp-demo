# 散戶買賣超張數 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 散戶買賣超張數 |
| 欄位分類 | 籌碼 |
| 代碼 | `retaildifference` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 台股 |

## 語法
> 散戶買張 ─ 散戶賣張。  

```pascal
Value1 = GetField("散戶買賣超張數");
Value1 = GetField("retaildifference");
```

---

## 說明
散戶買張 ─ 散戶賣張。

散戶買張是單一價格成交值小於50萬的合計買進張數，而散戶賣張則是同樣級距的合計賣出張數。

週頻率以上為期間累計值。
