# 控盤者買賣超張數 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 控盤者買賣超張數 |
| 欄位分類 | 籌碼 |
| 代碼 | `controllerdifference` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 台股 |

## 語法
> 控盤者買張 ─ 控盤者賣張。  

```pascal
Value1 = GetField("控盤者買賣超張數");
Value1 = GetField("controllerdifference");
```

---

## 說明
控盤者買張 ─ 控盤者賣張。

控盤者買張是單一價格成交值大於100萬之間的合計買進張數，而控盤者賣張則是同樣級距的合計賣出張數。

週頻率以上為期間累計值。
