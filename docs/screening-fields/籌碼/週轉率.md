# 週轉率 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 週轉率 |
| 欄位分類 | 籌碼 |
| 代碼 | `TurnOverRatio` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月、季、年 |
| 支援商品 | 台股 |

## 語法
> 股票交易量與流通股數的比例。  

```pascal
Value1 = GetField("週轉率");
Value1 = GetField("TurnOverRatio");
```

---

## 說明
股票交易量與流通股數的比例。

計算公式  : 當期成交量總和／發行張數 * 100%。
