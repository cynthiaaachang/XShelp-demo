# 內部人持股比例 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 內部人持股比例 |
| 欄位分類 | 籌碼 |
| 代碼 | `Insidersharesheldratio` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 公司內部人持股占公司股本的比例。  

```pascal
Value1 = GetField("內部人持股比例", "M");
Value1 = GetField("Insidersharesheldratio", "M");
```

---

## 說明
公司內部人持股占公司股本的比例。

計算公式為內部人持股張數／公司股本＊100%。

內部人統計的範圍包含董監以及公司內部的經理人。

資料每月更新一次。
