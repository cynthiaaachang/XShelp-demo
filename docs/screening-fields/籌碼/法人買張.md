# 法人買張 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 法人買張 |
| 欄位分類 | 籌碼 |
| 代碼 | `InvestorTotalBuy` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 台股 |

## 語法
> 三大法人(外資、投信、自營商)合計買進張數。  

```pascal
Value1 = GetField("法人買張");
Value1 = GetField("InvestorTotalBuy");
```

---

## 說明
三大法人(外資、投信、自營商)合計買進張數。
計算公式：外資買張＋投信買張＋自營商買張。

週頻率以上為期間累計值。
