# 董監持股 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 董監持股 |
| 欄位分類 | 籌碼 |
| 代碼 | `DirectorHeld` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股、類股指數 |

## 語法
> 董事與監察人的持股張數。  
> 類股與細產業商品計算方式為成分股加總。  

```pascal
Value1 = GetField("董監持股", "M");
Value1 = GetField("DirectorHeld", "M");
```

---

## 說明
董事與監察人的持股張數。
類股與細產業商品計算方式為成分股加總。
