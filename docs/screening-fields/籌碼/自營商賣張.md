# 自營商賣張 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 自營商賣張 |
| 欄位分類 | 籌碼 |
| 代碼 | `Dtotalsell` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 台股 |

## 語法
> 自營商賣出張數。  

```pascal
Value1 = GetField("自營商賣張");
Value1 = GetField("Dtotalsell");
```

---

## 說明
自營商賣出張數。

週頻率以上為期間累計值。
