# 自營商成本 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 自營商成本 |
| 欄位分類 | 籌碼 |
| 代碼 | `DBSCost` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月、季、半年、年 |
| 支援商品 | 台股 |

## 語法
> 系統估算的自營商持股成本。  

```pascal
Value1 = GetField("自營商成本");
Value1 = GetField("DBSCost");
```

---

## 說明
系統估算的自營商持股成本。

計算方式是透過每日自營商買張，賣張，以及當日的價格，估算出自營商的長期持有成本。

週頻率以上為統計期間的期底值。
