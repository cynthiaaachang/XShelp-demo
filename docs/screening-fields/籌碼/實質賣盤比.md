# 實質賣盤比 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 實質賣盤比 |
| 欄位分類 | 籌碼 |
| 代碼 | `RealShortRatio` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 台股 |

## 語法
> 實質賣出張數佔當日成交量的比例。  

```pascal
Value1 = GetField("實質賣盤比");
Value1 = GetField("RealShortRatio");
```

---

## 說明
實質賣出張數佔成交量的比例。

計算公式為：(成交量－融券賣出張數－資券相抵張數－融資賣出張數)／成交量 * 100％。

指數類型商品則是統計所有成分股的資料加總。
