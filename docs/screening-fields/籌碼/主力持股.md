# 主力持股 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 主力持股 |
| 欄位分類 | 籌碼 |
| 代碼 | `LeaderSharesHeld` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 台股 |

## 語法
> 系統估計的長期主力持股數量。  

```pascal
Value1 = GetField("主力持股");
Value1 = GetField("LeaderSharesHeld");
```

---

## 說明
系統估計的長期主力持股數量。

透過每日的主力買賣超，系統推估出主力目前的持股總數量。
