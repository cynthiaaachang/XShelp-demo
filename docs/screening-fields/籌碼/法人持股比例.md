# 法人持股比例 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 法人持股比例 |
| 欄位分類 | 籌碼 |
| 代碼 | `InvestorSharesHeldRatio` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 台股 |

## 語法
> 系統估算值，計算公式為：法人持股/流通在外張數 * 100%。  
> **單位為%，例如：持股比例若為20.5%，則回傳20.5。**  

```pascal
Value1 = GetField("法人持股比例");
Value1 = GetField("InvestorSharesHeldRatio");
```

---

## 說明
系統估算值，計算公式為：法人持股/流通在外張數 * 100%。

單位為%，例如：持股比例若為20.5%，則回傳20.5。

週頻率以上為統計期間的期底值。
