# 分公司淨買超金額家數 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 分公司淨買超金額家數 |
| 欄位分類 | 籌碼 |
| 代碼 | `BranchesOfNetBuyers` |
| 單位 | 家 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股、類股指數 |

## 語法
> 統計買超的券商分點家數。  

```pascal
Value1 = GetField("分公司淨買超金額家數", "D");
Value1 = GetField("BranchesOfNetBuyers", "D");
```

---

## 說明
統計買超的券商分點家數。

買超的定義是買進金額 - 賣出金額 > 0。

類股商品（細產業指標、主題指標）計算方式為統計成份股內符合買超的分點的集合家數。例如成份股#1的買超分點為a, b, 成份股#2的買超分點為b, c, 則類股商品的買超家數則為3。
