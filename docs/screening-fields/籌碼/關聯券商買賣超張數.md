# 關聯券商買賣超張數 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 關聯券商買賣超張數 |
| 欄位分類 | 籌碼 |

| 單位 | 張 |

## 語法
> 「關聯券商」買進張數 － 賣出張數。  

```pascal
Value1 = GetField("關聯券商買賣超張數", "D");
Value1 = GetField("CorrelationBrokerDifference", "D");
```

---

## 說明
「關聯券商」買進張數 － 賣出張數。

關聯券商是比對每週集保股權分散表內「千張大戶持股」的變化與當週主力分點的買賣超，來找出千張大戶所在的分點，這些分點就稱為這檔商品的關聯券商。

資料從2017/06開始提供。
