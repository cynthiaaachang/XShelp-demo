# 自營商自行買賣買張 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 自營商自行買賣買張 |
| 欄位分類 | 籌碼 |
| 代碼 | `DSelftotalbuy` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 台股 |

## 語法
> 自營商買進張數內歸屬於自行買賣的數量。  

```pascal
Value1 = GetField("自營商自行買賣買張");
Value1 = GetField("DSelftotalbuy");
```

---

## 說明
自營商買進張數內歸屬於自行買賣的數量。

週頻率以上為期間累計值。
