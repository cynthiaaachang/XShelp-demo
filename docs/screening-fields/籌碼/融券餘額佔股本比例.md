# 融券餘額佔股本比例 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 融券餘額佔股本比例 |
| 欄位分類 | 籌碼 |
| 代碼 | `Shortsaleremain／Capital` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 台股 |

## 語法
> 計算公式  : 融券餘額張數／發行張數*100%。  

```pascal
Value1 = GetField("融券餘額佔股本比例");
Value1 = GetField("Shortsaleremain／Capital");
```
