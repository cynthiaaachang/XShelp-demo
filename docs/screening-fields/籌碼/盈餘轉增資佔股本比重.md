# 盈餘轉增資佔股本比重 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 盈餘轉增資佔股本比重 |
| 欄位分類 | 籌碼 |
| 代碼 | `CaptialIncreaseFromEarning／Capital` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、選股、警示、交易、函數 |
| 可用頻率 | 年 |
| 支援商品 | 台股 |

## 語法
> 股本形成中，盈餘轉增資所佔的比重。  
> 計算公式  : 盈餘轉增資金額／總股本金額*100%。  

```pascal
Value1 = GetField("盈餘轉增資佔股本比重", "Y");
Value1 = GetField("CaptialIncreaseFromEarning／Capital", "Y");
```
