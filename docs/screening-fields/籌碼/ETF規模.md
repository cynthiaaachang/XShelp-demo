# ETF規模 – （選股欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | ETF規模 |
| 欄位分類 | 籌碼 |
| 代碼 | `ETFSize` |
| 單位 | 台股:元；美股:元(美元) |
| 格式 | 數值 |
| 支援腳本 | 選股、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股、16.US.E |

## 語法
> ETF的規模，單位是元/美元.  

```pascal
Value1 = GetField("ETF規模", "M");
Value1 = GetField("ETFSize", "M");
```
