# 跌停家數 – （報價欄位） <kbd>市場統計</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 跌停家數 |
| 欄位分類 | 市場統計 |
| 代碼 | `DownLimitSecs`<br>`即時跌停家數`<br>`q_DownLimitSecs` |
| 單位 | 家 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 大盤、類股指數 |

## 語法
> 指數成分股內目前跌停的家數。也可以使用GetField("跌停家數", "D")來表示。  

```pascal
Value1 = GetQuote("跌停家數");
Value1 = GetQuote("DownLimitSecs");
Value1 = GetQuote("即時跌停家數");
Value1 = q_DownLimitSecs
```

---

## 說明
指數成分股內目前跌停的家數。也可以使用GetField("跌停家數", "D")來表示。

大盤商品的統計會扣除權證商品。
