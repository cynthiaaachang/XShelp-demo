# 每股淨值 – （報價欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 每股淨值 |
| 欄位分類 | 財務 |
| 代碼 | `NetValuePerShare`<br>`q_NetValuePerShare` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、香港股票、大陸股票、美(股票) |

## 語法
> 最新一期財報的每股淨值。  

```pascal
Value1 = GetQuote("每股淨值");
Value1 = GetQuote("NetValuePerShare");
Value1 = q_NetValuePerShare
```

---

## 說明
最新一期財報的每股淨值。

台股採財務比率表的「每股淨值(F)(TSE公告數)」項目。
美股採財務比率表上的「每股淨值」項目。

台股，陸股，港股採用的是累計財報的數值，而美股則是單季財報的數值。
