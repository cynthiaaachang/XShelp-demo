# 每股盈餘 – （報價欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 每股盈餘 |
| 欄位分類 | 財務 |
| 代碼 | `CurrentEPS`<br>`即期每股盈餘`<br>`q_CurrentEPS` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、香港股票、大陸股票、美(股票) |

## 語法
> 最新一期財報的「每股盈餘」欄位。  

```pascal
Value1 = GetQuote("每股盈餘");
Value1 = GetQuote("CurrentEPS");
Value1 = GetQuote("即期每股盈餘");
Value1 = q_CurrentEPS
```

---

## 說明
最新一期財報的「每股盈餘」欄位。

美股採用的是損益表上的「基本每股盈餘(由淨利計算)」項目。

目前支援台股，港股，陸股以及美股。
