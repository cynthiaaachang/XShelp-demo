# 每股營收 – （報價欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 每股營收 |
| 欄位分類 | 財務 |
| 代碼 | `OpeRevenuePerShare`<br>`q_OpeRevenuePerShare` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、香港股票、大陸股票、美(股票) |

## 語法
> 最新一期財報的「每股營業額(元)」欄位。  

```pascal
Value1 = GetQuote("每股營收");
Value1 = GetQuote("OpeRevenuePerShare");
Value1 = q_OpeRevenuePerShare
```

---

## 說明
當期財報的「每股營業額(元)」欄位。

台股，陸股，港股採用的是累計財報的數值，而美股則是單季財報的數值。
