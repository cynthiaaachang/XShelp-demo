# 財報期別 – （報價欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 財報期別 |
| 欄位分類 | 財務 |
| 代碼 | `FinancialStatementsTime`<br>`q_FinancialStatementsTime` |
| 單位 | 日期 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、香港股票、大陸股票、美(股票) |

## 語法
> **最新一期財報的年月別。資料格式為YYYYMM的6碼數字。財報期別如果是季，則以3、6、9、12來表示1Q、2Q、3Q、4Q。財報期別如果是半年，則以6、12來代表H、2H。例如最新一期財報為2022/2Q，則回傳值為202206。**  

```pascal
Value1 = GetQuote("財報期別");
Value1 = GetQuote("FinancialStatementsTime");
Value1 = q_FinancialStatementsTime
```
