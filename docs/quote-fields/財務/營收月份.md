# 營收月份 – （報價欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 營收月份 |
| 欄位分類 | 財務 |
| 代碼 | `RevenueMonth`<br>`q_RevenueMonth` |
| 單位 | 日期 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、香港股票、大陸股票、美(股票) |

## 語法
> **最新一期營收的年月別。資料格式為YYYYMM的6碼數字，例如最新一期營收為2023/8月，則回傳值為202308。**  

```pascal
Value1 = GetQuote("營收月份");
Value1 = GetQuote("RevenueMonth");
Value1 = q_RevenueMonth
```
