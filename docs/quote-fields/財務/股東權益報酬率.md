# 股東權益報酬率 – （報價欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 股東權益報酬率 |
| 欄位分類 | 財務 |
| 代碼 | `CurrentROE`<br>`即期股東報酬率%`<br>`q_CurrentROE` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、香港股票、大陸股票、美(股票) |

## 語法
> **最新一期財報的「股東權益報酬率」(ROE)欄位。單位為％，例如股東權益報酬率為10%時則回傳10。**  

```pascal
Value1 = GetQuote("股東權益報酬率");
Value1 = GetQuote("CurrentROE");
Value1 = GetQuote("即期股東報酬率%");
Value1 = q_CurrentROE
```

---

## 說明
最新一期財報的「股東權益報酬率」(ROE)欄位。單位為％，例如股東權益報酬率為10%時則回傳10。

台股採用的是財務比率表的「ROE(A)─稅後」項目。
美股、陸股、港股採用的是財務比率表上的「股東權益報酬率」項目。

台股，陸股，港股採用的是累計財報的數值，而美股則是單季財報的數值。
