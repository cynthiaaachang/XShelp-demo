# 毛利率 – （報價欄位） <kbd>財務</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 毛利率 |
| 欄位分類 | 財務 |
| 代碼 | `GrossMarginRate`<br>`q_GrossMarginRate` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、香港股票、大陸股票、美(股票) |

## 語法
> **最新一期財報的「營業毛利率」欄位。單位為％，例如營業毛利率為25%時則回傳25。**  

```pascal
Value1 = GetQuote("毛利率");
Value1 = GetQuote("GrossMarginRate");
Value1 = q_GrossMarginRate
```

---

## 說明
最新一期財報的「營業毛利率」欄位。單位為％，例如營業毛利率為25%時則回傳25。

台股，陸股，港股採用的是累計財報的數值，而美股則是單季財報的數值。
