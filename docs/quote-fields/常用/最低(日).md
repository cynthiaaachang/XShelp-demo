# 最低(日) – （報價欄位） <kbd>常用</kbd> <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 最低(日) |
| 欄位分類 | 常用、價格 |
| 代碼 | `DailyLow`<br>`q_DailyLow` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 全部 |

## 語法
> 當日的最低價。也可以使用 GetField("Low", "D") 來表示。  

```pascal
Value1 = GetQuote("最低(日)");
Value1 = GetQuote("DailyLow");
Value1 = q_DailyLow
```
