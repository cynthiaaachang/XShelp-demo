# 成交時間 – （報價欄位） <kbd>常用</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 成交時間 |
| 欄位分類 | 常用 |
| 代碼 | `Time`<br>`q_Time` |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 全部 |

## 語法
> 最新一筆成交的時間。也可以使用 GetField("Time", "Tick") 來表示。  

```pascal
Value1 = GetQuote("成交時間");
Value1 = GetQuote("Time");
Value1 = q_Time
```

---

## 說明
最新一筆成交的時間。也可以使用 GetField("Time", "Tick") 來表示。

時間格式是hhmmss的6位數字，例如103025代表10點30分25秒。
