# 最高(日) – （報價欄位） <kbd>常用</kbd> <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 最高(日) |
| 欄位分類 | 常用、價格 |
| 代碼 | `DailyHigh`<br>`q_DailyHigh` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 全部 |

## 語法
> 當日的最高價。也可以使用 GetField("High", "D") 來表示。  

```pascal
Value1 = GetQuote("最高(日)");
Value1 = GetQuote("DailyHigh");
Value1 = q_DailyHigh
```
