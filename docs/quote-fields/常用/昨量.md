# 昨量 – （報價欄位） <kbd>常用</kbd> <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 昨量 |
| 欄位分類 | 常用、量能 |
| 代碼 | `PreTotalVolume`<br>`q_PreTotalVolume` |
| 單位 | 台股:張；大盤/指數:元；期貨/選擇權:口；其他市場股票:股 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 全部 |

## 語法
> 前一個交易日的盤中成交量，不包含零股以及鉅額交易。  
> 也可以使用 GetField("Volume", "D")[1] 來表示。  
> 數值單位依照商品類型而異。  

```pascal
Value1 = GetQuote("昨量");
Value1 = GetQuote("PreTotalVolume");
Value1 = q_PreTotalVolume
```
