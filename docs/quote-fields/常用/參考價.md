| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 參考價 |
| 欄位分類 | 常用、價格 |
| 代碼 | `RefPrice`<br>`q_RefPrice` |
| 單位 | 元 |

## 語法
> 當日的參考價。也可以使用 GetField("RefPrice", "D") 來表示。  

```pascal
Value1 = GetQuote("參考價");
Value1 = GetQuote("RefPrice");
Value1 = q_RefPrice
```

---

## 說明
當日的參考價。也可以使用 GetField("RefPrice", "D") 來表示。

一般而言是前一交易日的收盤價，如果股票商品有除權息的話，則是除權息後的參考價。

除權息的計算公式如下(實際數值以交易所公佈為主)：
1.除息參考價為股價－股息
2.除權參考價為股價／(1+配股率)
3.同時除權息參考價為 (股價-股息)／(1+配股率)
