# 賣出 – （報價欄位） <kbd>常用</kbd> <kbd>五檔統計</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 賣出 |
| 欄位分類 | 常用、五檔統計 |
| 代碼 | `Ask`<br>`q_Ask` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、期貨、選擇權、香港股票、大陸股票、美(股票) |

## 語法
> 目前委託簿內賣方最低的賣價。  

```pascal
Value1 = GetQuote("賣出");
Value1 = GetQuote("Ask");
Value1 = q_Ask
```

---

## 說明
目前委託簿內賣方最低的賣價。

如果是台期貨商品的話，則是取五檔委託簿上面最低的賣價以及衍生委託單的委賣價的最低者。
