# 委買 – （報價欄位） <kbd>五檔統計</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 委買 |
| 欄位分類 | 五檔統計 |
| 代碼 | `BestBidSize`<br>`BidSize`<br>`最佳委買`<br>`q_BestBidSize`<br>`q_BidSize` |
| 單位 | 台股:張；期貨/選擇權:口；其他市場股票:股 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、期貨、選擇權、香港股票、大陸股票、美(股票) |

## 語法
> 目前委託簿內買方最高買價的委託量。  
> 數值單位依照商品而異。  

```pascal
Value1 = GetQuote("委買");
Value1 = GetQuote("BestBidSize");
Value1 = GetQuote("BidSize");
Value1 = GetQuote("最佳委買");
Value1 = q_BestBidSize
Value1 = q_BidSize
```

---

## 說明
目前委託簿內買方最高買價的委託量。
數值單位依照商品而異。

如果是台期貨商品的話，則比對五檔委託簿上面最高買價以及衍生委託單的委買價：

- 如果五檔的最高買價比較高，則是取此買價的委託量，
- 如果衍生委託單的委買價比較高，則是取衍生委託單的委買量，
- 如果兩者相同，則是取五檔最高委買價的委託量以及衍生委託單的委買量的加總。
