# 委比 – （報價欄位） <kbd>五檔統計</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 委比 |
| 欄位分類 | 五檔統計 |
| 代碼 | `OrderRatio`<br>`q_OrderRatio` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、大盤、類股指數、期貨、選擇權、香港股票、大陸股票、美(股票) |

## 語法
> 目前委買賣差佔委買賣比重(%)。  
> **計算公式為(總委買－總委賣)／(總委買＋總委賣) * 100%。數值的單位為%，當委比為8%時，回傳值為8。**  

```pascal
Value1 = GetQuote("委比");
Value1 = GetQuote("OrderRatio");
Value1 = q_OrderRatio
```
