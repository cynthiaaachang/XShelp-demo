# 委買賣差 – （報價欄位） <kbd>五檔統計</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 委買賣差 |
| 欄位分類 | 五檔統計 |
| 代碼 | `BidAskDiff`<br>`q_BidAskDiff` |
| 單位 | 台股:張；大盤/類股:張；期貨/選擇權:口；其他市場股票:股 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、大盤、類股指數、期貨、選擇權、香港股票、大陸股票、美(股票) |

## 語法
> 目前總委買與總委賣的差異。  
> 計算公式 : 總委買－總委賣。  
> 數值單位依照商品而異。  

```pascal
Value1 = GetQuote("委買賣差");

Value1 = q_BidAskDiff
```
