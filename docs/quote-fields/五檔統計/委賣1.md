# 委賣1 – （報價欄位） <kbd>五檔統計</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 委賣1 |
| 欄位分類 | 五檔統計 |
| 代碼 | `BestAskSize1`<br>`q_BestAskSize1` |
| 單位 | 台股:張；期貨/選擇權:口；其他市場股票:股 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、期貨、選擇權、香港股票、大陸股票、美(股票) |

## 語法
> 委託簿上的第一檔賣出委託量。  
> 數值單位依照商品而異。  

```pascal
Value1 = GetQuote("委賣1");
Value1 = GetQuote("BestAskSize1");
Value1 = q_BestAskSize1
```

---

## 說明
委託簿上的第一檔賣出委託量。
數值單位依照商品而異。
