# 總委買 – （報價欄位） <kbd>五檔統計</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 總委買 |
| 欄位分類 | 五檔統計 |
| 代碼 | `SumBidSize`<br>`q_SumBidSize` |
| 單位 | 台股:張；大盤/類股:張；期貨/選擇權:口；其他市場股票:股 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、大盤、類股指數、期貨、選擇權、香港股票、大陸股票、美(股票) |

## 語法
> 委託簿上的前五檔委買數量的加總。  
> 數值單位依照商品而異。  

```pascal
Value1 = GetQuote("總委買");
Value1 = GetQuote("SumBidSize");
Value1 = q_SumBidSize
```

---

## 說明
委託簿上的前五檔委買數量的加總。
數值單位依照商品而異。

如果是大盤或是類股指數的話，數值是成分股所有商品的總委買張數的加總。
