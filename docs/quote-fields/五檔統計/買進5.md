# 買進5 – （報價欄位） <kbd>五檔統計</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 買進5 |
| 欄位分類 | 五檔統計 |
| 代碼 | `BestBid5`<br>`q_BestBid5` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、期貨、選擇權、香港股票、大陸股票、美(股票) |

## 語法
> 委託簿上的第五檔買進價。  

```pascal
Value1 = GetQuote("買進5");
Value1 = GetQuote("BestBid5");
Value1 = q_BestBid5
```
