# 暫緩撮合狀態 – （報價欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 暫緩撮合狀態 |
| 欄位分類 | 其他 |
| 代碼 | `CircuitBreakState`<br>`q_CircuitBreakState` |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台(股票)、期貨 |

## 語法
> **此欄位會回傳目前市場是否處於盤中價格穩定措施。**  

```pascal
Value1 = GetQuote("暫緩撮合狀態");
Value1 = GetQuote("CircuitBreakState");
Value1 = q_CircuitBreakState
```

---

## 說明
回傳值代表的意義：
0 = 無
1 = 暫緩搓合且趨跌
2 = 暫緩搓合且趨漲

證交所價格穩定措施的流程:

如果盤中發生價格穩定措施時，此時q_MarketState = 0，而q_CircuitBreakState會是1 或 2。
接下來q_MarketState會變成8，此時可以抓到試搓價格。
等到價格穩定措施結束後，q_MarketState 和CircuitBreakState 會變回0，商品恢復正常交易。

如果是開盤時/收盤時發生價格穩定措施的話，則q_MarketState會是7 或 3。
