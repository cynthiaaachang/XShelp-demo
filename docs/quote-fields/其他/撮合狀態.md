# 撮合狀態 – （報價欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 撮合狀態 |
| 欄位分類 | 其他 |
| 代碼 | `MarketState`<br>`q_MarketState` |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台(股票)、期貨 |

## 語法
> **此欄位會回傳目前的市場狀態。**  

```pascal
Value1 = GetQuote("撮合狀態");
Value1 = GetQuote("MarketState");
Value1 = q_MarketState
```

---

## 說明
以下為各種回傳值代表的意義：
- 0 = 正常交易
- 2 = 暫停交易
- 3 = 延緩收盤
- 4 = 開盤前試搓
- 5 = 開盤前試搓/不允許刪改單 (期交所)
- 6 = 收盤前試搓
- 7 = 暫緩開盤
- 8 = 暫停交易後開始試搓(證交所/期交所)
- 9 = 暫停交易後開始試搓/不允許刪改單 (期交所)
