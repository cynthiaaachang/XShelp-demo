# 試搓成交價 – （報價欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 試搓成交價 |
| 欄位分類 | 其他 |
| 代碼 | `SimulatedTradePrice`<br>`q_SimulatedTradePrice` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台(股票)、期貨 |

## 語法
> 試搓成交價格。  

```pascal
Value1 = GetQuote("試搓成交價");
Value1 = GetQuote("SimulatedTradePrice");
Value1 = q_SimulatedTradePrice
```

---

## 說明
此欄位會回傳試搓的成交價格。
