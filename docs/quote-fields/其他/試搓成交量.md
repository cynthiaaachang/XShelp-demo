# 試搓成交量 – （報價欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 試搓成交量 |
| 欄位分類 | 其他 |
| 代碼 | `SimulatedTradeVolume`<br>`q_SimulatedTradeVolume` |
| 單位 | 張/口 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台(股票)、期貨 |

## 語法
> 試搓成交數量(單位: 張/口)。  

```pascal
Value1 = GetQuote("試搓成交量");
Value1 = GetQuote("SimulatedTradeVolume");
Value1 = q_SimulatedTradeVolume
```

---

## 說明
此欄位會回傳試搓的成交數量。
