# 委賣均 – （報價欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 委賣均 |
| 欄位分類 | 量能 |
| 代碼 | `AvgShortUnits`<br>`q_AvgShortUnits` |
| 單位 | 大盤:張；期貨/選擇權:口 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 大盤、期貨、選擇權 |

## 語法
> 每一筆委賣的平均張數/口數。  
> 計算公式：累計委賣／累委賣筆。  

```pascal
Value1 = GetQuote("委賣均");
Value1 = GetQuote("AvgShortUnits");
Value1 = q_AvgShortUnits
```

---

## 說明
每一筆委賣的平均張數/口數。
計算公式：累計委賣／累委賣筆。
