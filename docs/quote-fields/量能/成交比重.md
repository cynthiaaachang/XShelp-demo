# 成交比重 – （報價欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 成交比重 |
| 欄位分類 | 量能 |
| 代碼 | `CashDirect`<br>`即時資金流向`<br>`q_CashDirect` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、類股指數 |

## 語法
> 成交值占對應大盤的成交比重(％)。  
> **單位是%，例如數值是5.6％的話，則回傳5.6。**  

```pascal
Value1 = GetQuote("成交比重");
Value1 = GetQuote("CashDirect");
Value1 = GetQuote("即時資金流向");
Value1 = q_CashDirect
```

---

## 說明
成交值占對應大盤的成交比重(％)。
單位是%，例如數值是5.6％的話，則回傳5.6。

支援台股以及類股指數，計算公式是成交值／對應大盤成交值＊100％。
