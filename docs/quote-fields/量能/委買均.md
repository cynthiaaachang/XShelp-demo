# 委買均 – （報價欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 委買均 |
| 欄位分類 | 量能 |
| 代碼 | `AvgLongUnits`<br>`q_AvgLongUnits` |
| 單位 | 大盤:張；期貨/選擇權:口 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 大盤、期貨、選擇權 |

## 語法
> 每一筆委買的平均張數/口數。  
> 計算公式：累計委買／累委買筆。  

```pascal
Value1 = GetQuote("委買均");
Value1 = GetQuote("AvgLongUnits");
Value1 = q_AvgLongUnits
```

---

每一筆委買的平均張數/口數。
