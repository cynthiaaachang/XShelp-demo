# 成交均量 – （報價欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 成交均量 |
| 欄位分類 | 量能 |
| 代碼 | `AvgDealedShare`<br>`q_AvgDealedShare` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 大盤 |

## 語法
> 整體市場(扣除權證)的每筆成交平均張數。  
> 計算公式：累計成交／累成交筆。  

```pascal
Value1 = GetQuote("成交均量");
Value1 = GetQuote("AvgDealedShare");
Value1 = q_AvgDealedShare
```
