# 累成交筆 – （報價欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 累成交筆 |
| 欄位分類 | 量能 |
| 代碼 | `CulMatchTicks`<br>`q_CulMatchTicks` |
| 單位 | 筆 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 大盤 |

## 語法
> 開盤迄今整體市場(扣除權證)的成交總筆數。也可以使用 GetField("累成交筆", "D") 來表示。  

```pascal
Value1 = GetQuote("累成交筆");
Value1 = GetQuote("CulMatchTicks");
Value1 = q_CulMatchTicks
```
