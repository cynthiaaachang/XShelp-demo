# 開盤買筆 – （報價欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 開盤買筆 |
| 欄位分類 | 量能 |
| 代碼 | `BoughtTickAtOpen`<br>`q_BoughtTickAtOpen` |
| 單位 | 筆 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 大盤、期貨、選擇權 |

## 語法
> 開盤時委託買進的筆數。也可以使用GetField("開盤買筆", "D")來表示。  

```pascal
Value1 = GetQuote("開盤買筆");
Value1 = GetQuote("BoughtTickAtOpen");
Value1 = q_BoughtTickAtOpen
```

---

## 說明
開盤時委託買進的筆數。也可以使用GetField("開盤買筆", "D")來表示。

大盤的數值會扣除權證。
