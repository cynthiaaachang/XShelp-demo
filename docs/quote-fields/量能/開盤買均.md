# 開盤買均 – （報價欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 開盤買均 |
| 欄位分類 | 量能 |
| 代碼 | `BoughtAverageAtOpen`<br>`q_BoughtAverageAtOpen` |
| 單位 | 大盤:張；期貨/選擇權:口 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 大盤、期貨、選擇權 |

## 語法
> 開盤時每筆委託買進張數/口數。  
> 計算公式：開盤委買／開盤買筆。  

```pascal
Value1 = GetQuote("開盤買均");
Value1 = GetQuote("BoughtAverageAtOpen");
Value1 = q_BoughtAverageAtOpen
```

---

## 說明
開盤時每筆委託買進張數/口數。
計算公式：開盤委買／開盤買筆。

大盤的數值是整體市場扣除權證後的數值。
