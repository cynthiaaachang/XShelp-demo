# 累計委買 – （報價欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 累計委買 |
| 欄位分類 | 量能 |
| 代碼 | `BidUnits`<br>`即時委買張數`<br>`q_BidUnits` |
| 單位 | 大盤:張；期貨/選擇權:口 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 大盤、期貨、選擇權 |

## 語法
> 開盤迄今的總委買張數/口數。也可以使用 GetField("累計委買", "D") 來表示。  

```pascal
Value1 = GetQuote("累計委買");
Value1 = GetQuote("BidUnits");
Value1 = GetQuote("即時委買張數");
Value1 = q_BidUnits
```

---

## 說明
開盤迄今的總委買張數/口數。也可以使用 GetField("累計委買", "D") 來表示。

證券市場的數值為扣除權證的統計。
