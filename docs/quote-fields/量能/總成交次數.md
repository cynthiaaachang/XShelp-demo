# 總成交次數 – （報價欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 總成交次數 |
| 欄位分類 | 量能 |
| 代碼 | `TotalTicks`<br>`q_TotalTicks` |
| 單位 | 筆 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、期貨、選擇權、香港股票、大陸股票、美(股票) |

## 語法
> 當日到目前為止的成交明細資料總筆數。也可以使用 GetField("總成交次數","D") 來表示。  

```pascal
Value1 = GetQuote("總成交次數");
Value1 = GetQuote("TotalTicks");
Value1 = q_TotalTicks
```
