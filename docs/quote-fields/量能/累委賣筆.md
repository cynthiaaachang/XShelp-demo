# 累委賣筆 – （報價欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 累委賣筆 |
| 欄位分類 | 量能 |
| 代碼 | `CulAskTicks`<br>`累計賣筆`<br>`q_CulAskTicks` |
| 單位 | 筆 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 大盤、期貨、選擇權 |

## 語法
> 開盤迄今的總委賣筆數。也可以使用 GetField("累委賣筆", "D") 來表示。  

```pascal
Value1 = GetQuote("累委賣筆");
Value1 = GetQuote("CulAskTicks");
Value1 = GetQuote("累計賣筆");
Value1 = q_CulAskTicks
```

## 說明
開盤迄今的總委賣筆數。也可以使用 GetField("累委賣筆", "D") 來表示。

證券市場的數值為扣除權證的統計。
