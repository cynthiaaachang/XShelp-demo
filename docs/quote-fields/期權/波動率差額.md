# 波動率差額 – （報價欄位） <kbd>期權</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 波動率差額 |
| 欄位分類 | 期權 |
| 代碼 | `VolatilityDiff`<br>`q_VolatilityDiff` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台(權證)、選擇權 |

## 語法
> 歷史波動率與隱含波動率的差異。  
> 計算公式：歷史波動率 ─ 隱含波動率。  

```pascal
Value1 = GetQuote("波動率差額");
Value1 = GetQuote("VolatilityDiff");
Value1 = q_VolatilityDiff
```

---

## 說明
歷史波動率與隱含波動率的差異。
計算公式：歷史波動率 ─ 隱含波動率。

例如歷史波動率為34.03%，隱含波動率為49.16，則波動率差額為-15.13%，回傳值為-15.13。
