# 內含值 – （報價欄位） <kbd>期權</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 內含值 |
| 欄位分類 | 期權 |
| 代碼 | `IntrinsicValue`<br>`q_IntrinsicValue` |
| 單位 | 點 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台(權證)、選擇權 |

## 語法
> 選擇權／權證商品的內含值(Intrinsic value)，也就是標的股現價與履約價的差異。  

```pascal
Value1 = GetQuote("內含值");
Value1 = GetQuote("IntrinsicValue");
Value1 = q_IntrinsicValue
```

---

## 說明
選擇權的內含值(Intrinsic value)，也就是標的股現價與履約價的差異。

如果是買權/認購權證的話，計算公式為MAX(標的股現價－履約價，0)＊執行比例。
如果是賣權/認售權證的話，計算公式為MAX(履約價－標的股現價，0)＊執行比例。
