# 最後交易日 – （報價欄位） <kbd>期權</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 最後交易日 |
| 欄位分類 | 期權 |
| 代碼 | `LastTradingDate`<br>`q_LastTradingDate` |
| 單位 | 日期 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台(權證)、期貨、選擇權 |

## 語法
> 商品的最後交易日。欄位格式為西元年月日，例如20221101。  

```pascal
Value1 = GetQuote("最後交易日");
Value1 = GetQuote("LastTradingDate");
Value1 = q_LastTradingDate
```
