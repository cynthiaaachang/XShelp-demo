# 買賣權成交量比率 – （報價欄位） <kbd>期權</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 買賣權成交量比率 |
| 欄位分類 | 期權 |
| 代碼 | `CPTradeRatio`<br>`q_CPTradeRatio` |
| 單位 | 比值 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 選擇權 |

## 語法
> 同契約所有賣權商品(PUT)的成交量／所有買權商品(CALL)的成交量。  

```pascal
Value1 = GetQuote("買賣權成交量比率");
Value1 = GetQuote("CPTradeRatio");
Value1 = q_CPTradeRatio
```

---

## 說明
同契約所有賣權商品(PUT)的成交量／所有買權商品(CALL)的成交量。
