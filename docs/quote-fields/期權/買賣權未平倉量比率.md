# 買賣權未平倉量比率 – （報價欄位） <kbd>期權</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 買賣權未平倉量比率 |
| 欄位分類 | 期權 |
| 代碼 | `CPOIRatio`<br>`q_CPOIRatio` |
| 單位 | 比值 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 選擇權 |

## 語法
> 同契約所有賣權商品(PUT)的未平倉量／買權商品(CALL)的未平倉量。  

```pascal
Value1 = GetQuote("買賣權未平倉量比率");
Value1 = GetQuote("CPOIRatio");
Value1 = q_CPOIRatio
```

---

## 說明
同契約所有賣權商品(PUT)的未平倉量／買權商品(CALL)的未平倉量。
