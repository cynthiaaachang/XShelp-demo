# 履約價 – （報價欄位） <kbd>期權</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 履約價 |
| 欄位分類 | 期權 |
| 代碼 | `StrikePrice`<br>`q_StrikePrice` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台(權證)、選擇權 |

## 語法
> 選擇權／權證商品的履約價。也可以使用 GetSymbolInfo("履約價") 來表示。  

```pascal
Value1 = GetQuote("履約價");
Value1 = GetQuote("StrikePrice");
Value1 = q_StrikePrice
```
