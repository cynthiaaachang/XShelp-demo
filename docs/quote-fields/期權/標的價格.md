# 標的價格 – （報價欄位） <kbd>期權</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 標的價格 |
| 欄位分類 | 期權 |
| 代碼 | `TargetPrice`<br>`q_TargetPrice` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台(權證)、選擇權 |

## 語法
> 選擇權／權證商品的標的股目前的價格。  

```pascal
Value1 = GetQuote("標的價格");
Value1 = GetQuote("TargetPrice");
Value1 = q_TargetPrice
```

---

## 說明
選擇權／權證商品的標的股目前的價格。
