# 損益兩平 – （報價欄位） <kbd>期權</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 損益兩平 |
| 欄位分類 | 期權 |
| 代碼 | `BreakEvenPoint`<br>`q_BreakEvenPoint` |
| 單位 | 值 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台(權證) |

## 語法
> 讓權證商品可以達到損益兩平的標的股價格。  

```pascal
Value1 = GetQuote("損益兩平");
Value1 = GetQuote("BreakEvenPoint");
Value1 = q_BreakEvenPoint
```

---

## 說明
讓權證商品可以達到損益兩平的標的股價格。

認購權證的損益兩平價格是履約價＋權證價格 /執行比率。
認售權證的損益兩平價格則是履約價 ─ 權證價格 /執行比率。

舉例而言，假如認購權證的履約價是100，目前權證價格為1，而執行比例是0.2的話，那麼損益兩平的價格是100＋1／0.2＝105。
