# 股本(億) – （報價欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 股本(億) |
| 欄位分類 | 基本 |
| 代碼 | `CurrentCapitalin100Million`<br>`億元股本`<br>`q_CurrentCapitalin100Million` |
| 單位 | 億元 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、香港股票、大陸股票 |

## 語法
> 股票的股本，換算成億元。  

```pascal
Value1 = GetQuote("股本(億)");
Value1 = GetQuote("CurrentCapitalin100Million");
Value1 = GetQuote("億元股本");
Value1 = q_CurrentCapitalin100Million
```

---

## 說明
股票的股本，換算成億元。
