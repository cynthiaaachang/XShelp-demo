# 發行股數 – （報價欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 發行股數 |
| 欄位分類 | 基本 |
| 代碼 | `CurrentShareCapital`<br>`q_CurrentShareCapital` |
| 單位 | 股 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、香港股票、大陸股票 |

## 語法
> 最新一期的發行股數。  
> 目前支援台股，陸股以及港股。  

```pascal
Value1 = GetQuote("發行股數");
Value1 = GetQuote("CurrentShareCapital");
Value1 = q_CurrentShareCapital
```

---

## 說明
最新一期的發行股數。
目前支援台股，陸股，港股三個市場。

如果是台股的話，則按「股份有限公司」章程所訂之股份總數，依公司法第156條第2項之規定，得分次發行。是以，「股份有限公司」登記事項有「資本總額」（股份總數X每股金額）及「實收資本額」（已發行股份總數X每股金額）之區分。 此處的發行股數，即「實收資本額」中的「已發行股份總數」，目前台灣市場多數普通股的「每股金額」，都是10元，所以發行股數=實收資本額除以10。 如果一家上市公司的實收資本額是10億元，則 發行股數= 10億除以10，等於1億股。

如果是陸股的話，欄位數值為流通在外股數。
