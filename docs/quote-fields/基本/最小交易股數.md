# 最小交易股數 – （報價欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 最小交易股數 |
| 欄位分類 | 基本 |
| 代碼 | `MinTradingShares`<br>`q_MinTradingShares` |
| 單位 | 股 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、香港股票 |

## 語法
> 交易的最小單位股數。也可以使用 GetSymbolInfo("最小交易股數") 來表示。  

```pascal
Value1 = GetQuote("最小交易股數");
Value1 = GetQuote("MinTradingShares");
Value1 = q_MinTradingShares
```

---

## 說明
交易的最小單位股數。也可以使用 GetSymbolInfo("最小交易股數") 來表示。 

例如台股一般為1000股，但國外成分證券ETF類的商品可能不同，例如國泰日經225+U(美元)(00657K)為100股。 

香港股票則依照交易所定義，每檔商品不同。 

目前支援台股以及港股。
