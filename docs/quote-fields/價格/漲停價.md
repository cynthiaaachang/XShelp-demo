# 漲停價 – （報價欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 漲停價 |
| 欄位分類 | 價格 |
| 代碼 | `DailyUplimit`<br>`今日漲停價`<br>`q_DailyUplimit` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、期貨、選擇權、大陸股票 |

## 語法
> 當日的漲停價。也可以使用 GetField("漲停價", "D") 來表示。  

```pascal
Value1 = GetQuote("漲停價");
Value1 = GetQuote("DailyUplimit");
Value1 = GetQuote("今日漲停價");
Value1 = q_DailyUplimit
```
