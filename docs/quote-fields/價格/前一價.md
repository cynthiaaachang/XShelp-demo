# 前一價 – （報價欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 前一價 |
| 欄位分類 | 價格 |
| 代碼 | `PreMatch1`<br>`q_PreMatch1` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 全部 |

## 語法
> 最新一筆成交價的前一筆成交價格。也可以使用 GetField("Close", "Tick")[1] 來表示。  

```pascal
Value1 = GetQuote("前一價");
Value1 = GetQuote("PreMatch1");
Value1 = q_PreMatch1
```
