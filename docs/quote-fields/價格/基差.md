# 基差 – （報價欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 基差 |
| 欄位分類 | 價格 |
| 代碼 | `Basis`<br>`q_Basis` |
| 單位 | 點 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 大盤、期貨 |

## 語法
> 現貨價格與期貨價格之間的差異。也可以使用 GetField("基差") 來表示。  

```pascal
Value1 = GetQuote("基差");
Value1 = GetQuote("Basis");
Value1 = q_Basis
```

---

## 說明
現貨價格與期貨價格之間的差異。也可以使用 GetField("基差") 來表示。

計算公式：期貨基差＝現貨價格－期貨價格。

例如：加權指數為16800，而大台指為16795的話，那麼基差＝16800 ─ 16795＝5。
