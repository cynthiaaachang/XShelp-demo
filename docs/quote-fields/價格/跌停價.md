# 跌停價 – （報價欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 跌停價 |
| 欄位分類 | 價格 |
| 代碼 | `DailyDownlimit`<br>`今日跌停價`<br>`q_DailyDownlimit` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、期貨、選擇權、大陸股票 |

## 語法
> 當日的跌停價。也可以使用 GetField("跌停價", "D") 來表示。  

```pascal
Value1 = GetQuote("跌停價");
Value1 = GetQuote("DailyDownlimit");
Value1 = GetQuote("今日跌停價");
Value1 = q_DailyDownlimit
```
