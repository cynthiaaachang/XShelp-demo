# 價差 – （報價欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 價差 |
| 欄位分類 | 價格 |
| 代碼 | `Spread`<br>`q_Spread` |
| 單位 | 點 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 大盤、期貨 |

## 語法
> 期貨價格與現貨價格之間的差額。  

```pascal
Value1 = GetQuote("價差");
Value1 = GetQuote("Spread");
Value1 = q_Spread
```

---

## 說明
期貨價格與現貨價格之間的差額。

計算公式：期貨價差＝期貨價格－現貨價格。

例如：加權指數為16800，而大台指為16795的話，那麼基差＝16795 ─ 16800＝-5。
