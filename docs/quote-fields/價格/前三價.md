# 前三價 – （報價欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 前三價 |
| 欄位分類 | 價格 |
| 代碼 | `PreMatch3`<br>`q_PreMatch3` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 全部 |

## 語法
> 最新一筆成交價的前第三筆成交價格。也可以使用 GetField("Close", "Tick")[3] 來表示。  

```pascal
Value1 = GetQuote("前三價");
Value1 = GetQuote("PreMatch3");
Value1 = q_PreMatch3
```
