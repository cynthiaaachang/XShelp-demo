# 均價 – （報價欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 均價 |
| 欄位分類 | 價格 |
| 代碼 | `AvgPrice`<br>`q_AvgPrice` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股、大盤、類股指數、台(權證)、期貨、選擇權、香港股票、大陸股票、美(股票) |

## 語法
> 當日的平均成交價。也可以使用 GetField("均價", "D") 來表示。  

```pascal
Value1 = GetQuote("均價");
Value1 = GetQuote("AvgPrice");
Value1 = q_AvgPrice
```

---

## 說明
當日的平均成交價。也可以使用 GetField("均價", "D") 來表示。

計算方式是累加開盤後每筆成交的價格乘上數量，然後除以累計的成交數量，得出當日平均的成交價。
