# 振幅 – （報價欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 振幅 |
| 欄位分類 | 價格 |
| 代碼 | `DayAmplitude`<br>`當日振幅`<br>`q_DayAmplitude` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 全部 |

## 語法
> 當日的振幅。  

```pascal
Value1 = GetQuote("振幅");
Value1 = GetQuote("DayAmplitude");
Value1 = GetQuote("當日振幅");
Value1 = q_DayAmplitude
```

---

## 說明
當日的振幅。

振幅計算公式 = (當日最高價 - 當日最低價)／參考價＊100，數值單位為%，若振幅為2.5%，則回傳2.5。
