# 漲跌幅 – （報價欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 漲跌幅 |
| 欄位分類 | 價格 |
| 代碼 | `PriceChangeRatio`<br>`當日漲幅%`<br>`q_PriceChangeRatio` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 全部 |

## 語法
> 當日的漲跌幅。  

```pascal
Value1 = GetQuote("漲跌幅");
Value1 = GetQuote("PriceChangeRatio");
Value1 = GetQuote("當日漲幅%");
Value1 = q_PriceChangeRatio
```

---

## 說明
當日的漲跌幅。

漲跌幅計算方式為(最新一筆成交價 ─ 參考價)／參考價 ＊ 100。數值的單位是%，當漲跌幅為7%時，回傳7。
