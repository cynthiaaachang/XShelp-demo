# 撮合群組 – （資料欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 撮合群組 |
| 欄位分類 | 其他 |
| 代碼 | `TickGroup`<br>`撮合序號` |
| 單位 | 數值 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 1 |
| 支援商品 | 大盤、類股指數、台(股票) |

## 語法
> 這一筆成交明細的撮合方式。  

```pascal
Value1 = GetField("撮合群組", "Tick");
Value1 = GetField("TickGroup", "Tick");
Value1 = GetField("撮合序號", "Tick");
```

---

## 說明
這一筆成交明細的撮合方式。

僅支援台股商品。其他商品類型都會回傳-1。

此欄位會依照證交所所提供的成交序列資料，回傳以下可能的數值：

-1: 表示這一筆成交不是逐筆撮合，例如開盤的第一筆成交就會標示-1，

0: 表示這一筆成交是逐筆撮合，可是這一次逐筆撮合只有產生一筆成交紀錄，

1: 表示這一筆成交是逐筆撮合，可是這一次逐筆撮合產生了多筆的成交記錄，而這一筆成交是這一批的第一筆，

2: 表示這一筆成交是逐筆撮合，可是這一次逐筆撮合產生了多筆的成交記錄，而這一筆成交是這一批中間的資料(不是第一筆也不是最後一筆)，

3 表示這一筆成交是逐筆撮合，可是這一次逐筆撮合產生了多筆的成交記錄，而這一筆成交是這一批的最後一筆。

詳細的定義，請參考 https://www.xq.com.tw/learn/xspractice/tickseq/。
