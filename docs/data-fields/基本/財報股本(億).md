# 財報股本(億) – （資料欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 財報股本(億) |
| 欄位分類 | 基本 |
| 代碼 | `CurrentCapitalin100Million`<br>`億元股本`<br>`加權平均股本` |
| 單位 | 億元 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 季、半年、年 |
| 支援商品 | 台股 |

## 語法
> 公司財報(季、年)上載明的股本，換算成億元。  

```pascal
Value1 = GetField("財報股本(億)");
Value1 = GetField("CurrentCapitalin100Million");
Value1 = GetField("億元股本");
Value1 = GetField("加權平均股本");
```

---

## 說明
公司財報(季、年)上載明的股本，換算成億元。
