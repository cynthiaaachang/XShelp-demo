# 總市值(元) – （資料欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 總市值(元) |
| 欄位分類 | 基本 |
| 代碼 | `TotalMarketValue`<br>`總市值` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、季、半年、年 |
| 支援商品 | 台股、大盤、類股指數 |

## 語法
> 股票或是指數的市值。股票市值以發行股數 ＊ 收盤價來計算，指數則是成分股的市值加總。單位為元。  

```pascal
Value1 = GetField("總市值(元)");
Value1 = GetField("TotalMarketValue");
Value1 = GetField("總市值");
```

---

## 說明
股票或是指數的市值。股票市值以發行股數 ＊ 收盤價來計算，指數則是成分股的市值加總。單位為元。

週以上頻率，是取期底值。
