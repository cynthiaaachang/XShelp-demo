# 月營收 – （資料欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 月營收 |
| 欄位分類 | 基本 |
| 代碼 | `Sales` |
| 單位 | 億 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股、大盤 |

## 語法
> 每個月的營業收入。  

```pascal
Value1 = GetField("月營收", "M");
Value1 = GetField("Sales", "M");
```

## 說明
每個月的營業收入。
