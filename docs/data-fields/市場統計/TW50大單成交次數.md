## 說明
加總台灣50成分股每檔商品最近10分鐘的買進大單+買進特大單的平均次數。

僅支援TSE50.SJ 商品。

統計的方式不跨日，所以開盤未滿10分鐘前則依照開盤的分鐘數來平均。

建議撰寫方式：
value1 = getsymbolfield("TSE50.SJ","TW50大單成交次數");
