# 外盤家數 – （資料欄位） <kbd>市場統計</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 外盤家數 |
| 欄位分類 | 市場統計 |
| 代碼 | `AskSecurities` |
| 單位 | 家 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 分鐘、日 |
| 支援商品 | 大盤、類股指數 |

## 語法
> 指數成分股內目前以外盤成交的家數。  

```pascal
Value1 = GetField("外盤家數");
Value1 = GetField("AskSecurities");
```

---

## 說明
指數成分股內目前以外盤成交的家數。

統計資料盤中會定時更新，如果是分鐘頻率的話會取得當下最新數值，如果是日頻率的話則是當日最後的數值。

大盤商品的統計會扣除權證商品。
