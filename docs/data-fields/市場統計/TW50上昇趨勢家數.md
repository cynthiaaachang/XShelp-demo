## 說明
統計台灣50成份股內趨勢向上的家數。
趨勢向上的定義是計算近6分鐘(含目前這一分鐘)的線性回歸線(LinearRegression)是否向上。

僅支援TSE50.SJ 商品。

系統每分鐘會統計符合的家數，而腳本端則依照執行頻率(支援分鐘/日頻率)來取得當分鐘最後的符合家數。

建議撰寫方式：
value1 = getsymbolfield("TSE50.SJ","TW50上昇趨勢家數");
