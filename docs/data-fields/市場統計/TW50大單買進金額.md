# TW50大單買進金額 – （資料欄位） <kbd>市場統計</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | TW50大單買進金額 |
| 欄位分類 | 市場統計 |
| 代碼 | `TW50BidVolumePriceXLTrend` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 分鐘、日 |
| 支援商品 | 大盤 |

## 語法
> 加總台灣50成分股每檔商品的分鐘買進大單+買進特大單的金額。  
> 分鐘買進金額為開盤迄今最近10分鐘的分鐘均量。  
> 僅支援TSE50.SJ 商品。  

```pascal
Value1 = GetField("TW50大單買進金額");
Value1 = GetField("TW50BidVolumePriceXLTrend");
```

---

## 說明
加總台灣50成分股每檔商品的分鐘買進大單+買進特大單的金額。

分鐘買進金額為開盤迄今最近10分鐘的分鐘均量。

僅支援TSE50.SJ 商品。

統計的方式不跨日，所以開盤未滿10分鐘前則依照開盤的分鐘數來平均。

建議撰寫方式：
value1 = getsymbolfield("TSE50.SJ","TW50大單買進金額");
