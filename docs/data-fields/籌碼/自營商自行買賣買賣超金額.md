# 自營商自行買賣買賣超金額 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 自營商自行買賣買賣超金額 |
| 欄位分類 | 籌碼 |
| 代碼 | `DSelfDifferenceAmount` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日 |
| 支援商品 | 大盤 |

## 語法
> 計算公式為自營商自行買賣買進金額 ─ 自營商自行買賣賣出金額。  

```pascal
Value1 = GetField("自營商自行買賣買賣超金額", "D");
Value1 = GetField("DSelfDifferenceAmount", "D");
```

---

## 說明
計算公式為自營商自行買賣買進金額 ─ 自營商自行買賣賣出金額。

週頻率以上為期間累計值。
