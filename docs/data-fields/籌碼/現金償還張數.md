# 現金償還張數 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 現金償還張數 |
| 欄位分類 | 籌碼 |
| 代碼 | `POMRC` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、還原日、還原週、還原月、季、半年、年 |
| 支援商品 | 台股、大盤 |

## 語法
> 以現金方式償還融資的張數。  
> 交易所公告的融資的現金償還張數。  
> 週頻率以上為期間累計值。  
> 每日下午10:30後開始更新。  

```pascal
Value1 = GetField("現金償還張數");
Value1 = GetField("POMRC");
```

---

## 說明
以現金方式償還融資的張數。

現金償還指的是以補足現金把融資股票轉成現股。

週頻率以上為期間累計值。
