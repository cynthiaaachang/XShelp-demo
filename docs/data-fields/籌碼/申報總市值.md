# 申報總市值 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 申報總市值 |
| 欄位分類 | 籌碼 |
| 代碼 | `DTTotalValue` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 大盤、類股指數 |

## 語法
> 內部人申報轉讓持股的市值加總。  

```pascal
Value1 = GetField("申報總市值");
Value1 = GetField("DTTotalValue");
```

---

## 說明
內部人申報轉讓持股的市值加總。

申報轉讓市值的計算方式是轉讓張數乘上當日收盤價。

類股商品（細產業指標、主題指標）計算方式為成分股加總。

週頻率以上為期間累計值。
