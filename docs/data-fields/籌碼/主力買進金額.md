# 主力買進金額 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 主力買進金額 |
| 欄位分類 | 籌碼 |
| 代碼 | `MBAmount` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日 |
| 支援商品 | 大盤 |

## 語法
> 前十五大買超分點的買超金額合計。  

```pascal
Value1 = GetField("主力買進金額", "D");
Value1 = GetField("MBAmount", "D");
```

---

## 說明
前十五大買超分點的買超金額合計。

如果是個股的話，數值為前15大主力買進券商當日買超金額(買進金額─賣超金額)的加總。

如果是大盤的話，則依照每個券商分點對所有成分股的買超張數(買進張數─賣超張數)找出前15大分點，然後加總這些分點對所有成分股的買超金額。
