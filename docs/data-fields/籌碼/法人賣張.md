# 法人賣張 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 法人賣張 |
| 欄位分類 | 籌碼 |
| 代碼 | `InvestorTotalSell` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、還原日、還原週、還原月、季、半年、年 |
| 支援商品 | 台股 |

## 語法
> 三大法人(外資、投信、自營商)合計賣出張數。  

```pascal
Value1 = GetField("法人賣張");
Value1 = GetField("InvestorTotalSell");
```

---

## 說明
三大法人(外資、投信、自營商)合計賣出張數。
計算公式：外資賣張＋投信賣張＋自營商賣張。

週頻率以上為期間累計值。
