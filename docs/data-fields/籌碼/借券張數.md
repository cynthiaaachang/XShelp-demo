# 借券張數 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 借券張數 |
| 欄位分類 | 籌碼 |
| 代碼 | `SBLBorrowing` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、還原日、還原週、還原月、季、半年、年 |
| 支援商品 | 台股、大盤 |

## 語法
> 借券的張數。  

```pascal
Value1 = GetField("借券張數");
Value1 = GetField("SBLBorrowing");
```

---

## 說明
借券的張數。

週頻率以上為期間累計值。

借券數量係指當日投資人透過證交所借券中心或向證券商、證金公司借入之證券數量。這個數量並不一定等於借券放空的數量。
