# 官股券商買進金額 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 官股券商買進金額 |
| 欄位分類 | 籌碼 |
| 代碼 | `OBBAmount` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |

## 語法
> 「官股券商」對市場所有股票的合計買進金額。  

```pascal
Value1 = GetField("官股券商買進金額", "D");
Value1 = GetField("OBBAmount", "D");
```

---

## 說明
「官股券商」對市場所有股票的合計買進金額。

官股券商名單為: 臺銀，土銀，合庫，兆豐，第一金，華南永昌，彰銀以及台企銀。

週頻率以上為期間累計值。
