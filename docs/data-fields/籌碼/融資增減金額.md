# 融資增減金額 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 融資增減金額 |
| 欄位分類 | 籌碼 |
| 代碼 | `IndexPOMDiff` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、季、半年、年 |
| 支援商品 | 大盤 |

## 語法
> 融資餘額(金額)與前一期的異動金額。  

```pascal
Value1 = GetField("融資增減金額");
Value1 = GetField("IndexPOMDiff");
```

---

## 說明
融資餘額(金額)與前一期的異動金額。

融資餘額金額為到目前為止以融資買進的淨金額(元)。

週頻率以上為期間累計值。
