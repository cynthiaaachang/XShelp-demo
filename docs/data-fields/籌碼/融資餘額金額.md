# 融資餘額金額 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 融資餘額金額 |
| 欄位分類 | 籌碼 |
| 代碼 | `IndexPOMRemain` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、季、半年、年 |
| 支援商品 | 大盤 |

## 語法
> 融資餘額金額，也就是到目前以融資買進的淨金額。  

```pascal
Value1 = GetField("融資餘額金額");
Value1 = GetField("IndexPOMRemain");
```

---

## 說明
融資餘額金額，也就是到目前以融資買進的淨金額。

這個數值代表著還有多少融資金額尚未償還。

週頻率以上為統計期間的期底值。
