# 法人賣出金額 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 法人賣出金額 |
| 欄位分類 | 籌碼 |
| 代碼 | `InvestorTotalSellAmount` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、季、半年、年 |
| 支援商品 | 大盤 |

## 語法
> 三大法人(外資、投信、自營商)合計賣出金額。  

```pascal
Value1 = GetField("法人賣出金額");
Value1 = GetField("InvestorTotalSellAmount");
```

---

## 說明
三大法人(外資、投信、自營商)合計賣出金額。
計算公式：外資賣出金額＋投信賣出金額＋自營商賣出金額。

週頻率以上為期間累計值。
