# 主力成本 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 主力成本 |
| 欄位分類 | 籌碼 |
| 代碼 | `LeaderCost` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、還原日、還原週、還原月、季、半年、年 |
| 支援商品 | 台股 |

## 語法
> 系統估計的長期主力持股成本。  

```pascal
Value1 = GetField("主力成本");
Value1 = GetField("LeaderCost");
```

---

## 說明
系統估計的長期主力持股成本。

透過每日的主力買賣超以及買進賣出成本，系統推估出主力目前的持股成本。
