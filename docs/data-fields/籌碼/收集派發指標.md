# 收集派發指標 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 收集派發指標 |
| 欄位分類 | 籌碼 |
| 代碼 | `ADIndicator` |
| 單位 | 家 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、還原日、還原週、還原月、季、半年、年 |
| 支援商品 | 台股 |

## 語法
> 分公司賣出家數 － 分公司買進家數。  

```pascal
Value1 = GetField("收集派發指標");
Value1 = GetField("ADIndicator");
```

---

## 說明
分公司賣出家數 － 分公司買進家數。

當賣出家數大於買進家數時(數值>0)，表示籌碼由多數人流向少數人手中，籌碼趨於集中，反之則表示籌碼趨於發散。

週頻率以上為期間累計值。
