# 現股當沖張數 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 現股當沖張數 |
| 欄位分類 | 籌碼 |
| 代碼 | `NormalDayTrades` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、還原日、還原週、還原月、季、半年、年 |
| 支援商品 | 台股、大盤、類股指數 |

## 語法
> 現股當日沖銷的張數。  

```pascal
Value1 = GetField("現股當沖張數");
Value1 = GetField("NormalDayTrades");
```

---

## 說明
現股當日沖銷的張數。

類股商品（細產業指標、主題指標）的數值為成分股現股當沖張數的加總。
大盤商品的數值為交易所公布值。

週頻率以上為期間累計值。
