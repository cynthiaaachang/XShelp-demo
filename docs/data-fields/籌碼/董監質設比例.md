# 董監質設比例 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 董監質設比例 |
| 欄位分類 | 籌碼 |
| 代碼 | `DSmortgageratio` |
| 單位 | % |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 董監持股內質設的比例。  

```pascal
Value1 = GetField("董監質設比例", "M");
Value1 = GetField("DSmortgageratio", "M");
```

---

## 說明
董監持股內質設的比例。

計算公式：董監持股內目前質設的張數／董監持股＊100%。

資料每月更新一次。
