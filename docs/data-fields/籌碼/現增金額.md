# 現增金額 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 現增金額 |
| 欄位分類 | 籌碼 |
| 代碼 | `PriceofCashIncrement` |
| 單位 | 億 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 語法
> 最新一期現金增資的發行總金額。  

```pascal
Value1 = GetField("現增金額");
Value1 = GetField("PriceofCashIncrement");
```

---

## 說明
最新一期現金增資的發行總金額。

可以檢查GetFieldDate("現增金額")是否等於0來判斷是否有現金增資的資料。
