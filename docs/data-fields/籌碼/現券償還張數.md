# 現券償還張數 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 現券償還張數 |
| 欄位分類 | 籌碼 |
| 代碼 | `ShortSalesRS` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、還原日、還原週、還原月、季、半年、年 |
| 支援商品 | 台股、大盤 |

## 語法
> 以現股方式償還融券的張數。  

```pascal
Value1 = GetField("現券償還張數");
Value1 = GetField("ShortSalesRS");
```

---

## 說明
以現股方式償還融券的張數。

現券償還指的是以現股來償還融券放空的部位。

週頻率以上為期間累計值。
