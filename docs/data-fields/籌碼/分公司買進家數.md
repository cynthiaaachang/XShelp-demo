# 分公司買進家數 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 分公司買進家數 |
| 欄位分類 | 籌碼 |
| 代碼 | `LongBranches` |
| 單位 | 家 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股 |

## 語法
> 統計買進金額大於零的券商分點家數。  

```pascal
Value1 = GetField("分公司買進家數", "D");
Value1 = GetField("LongBranches", "D");
```

---

## 說明
統計買進金額大於零的券商分點家數。

請注意：只要有買進的券商分點就會納入統計，與該分點是否買超無關。
