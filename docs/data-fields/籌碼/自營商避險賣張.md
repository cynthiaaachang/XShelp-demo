# 自營商避險賣張 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 自營商避險賣張 |
| 欄位分類 | 籌碼 |
| 代碼 | `DtotalHedgesell` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、季、半年、年 |
| 支援商品 | 台股、大盤 |

## 語法
> 自營商賣出張數內歸屬於避險的數量。  

```pascal
Value1 = GetField("自營商避險賣張");
Value1 = GetField("DtotalHedgesell");
```

---

## 說明
自營商買進張數內歸屬於避險的數量。

週頻率以上為期間累計值。
