# 董監持股佔股本比例 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 董監持股佔股本比例 |
| 欄位分類 | 籌碼 |
| 代碼 | `DirectorHeld／Capital` |
| 單位 | % |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 語法
> 董監持股占公司股本的比例。  

```pascal
Value1 = GetField("董監持股佔股本比例", "M");
Value1 = GetField("DirectorHeld／Capital", "M");
```

---

## 說明
董監持股占公司股本的比例。

計算公式為董監持股張數／公司股本＊100%。

董監持股統計的範圍包含公司內的董事以及監察人。

資料每月更新一次。
