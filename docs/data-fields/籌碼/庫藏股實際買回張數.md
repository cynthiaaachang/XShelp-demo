# 庫藏股實際買回張數 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 庫藏股實際買回張數 |
| 欄位分類 | 籌碼 |
| 代碼 | `Stockbuyvalue` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 語法
> 最新一次庫藏股公告的實際買回張數。  

```pascal
Value1 = GetField("庫藏股實際買回張數");
Value1 = GetField("Stockbuyvalue");
```

---

## 說明
最新一次庫藏股公告的實際買回張數。

使用時可以搭配庫藏股開始日期，庫藏股結束日期，以及庫藏股預計買回張數等欄位。
