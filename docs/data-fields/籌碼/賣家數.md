# 賣家數 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 賣家數 |
| 欄位分類 | 籌碼 |
| 代碼 | `NetShortBranches` |
| 單位 | 家 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股 |

## 語法
> 統計賣超張數大於零的券商家數。  

```pascal
Value1 = GetField("賣家數", "D");
Value1 = GetField("NetShortBranches", "D");
```

---

## 說明
統計賣超張數大於零的券商家數。

同一個券商的所有分點加總統計，如果合計為賣超，則算為1家。
