# 投信買張 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 投信買張 |
| 欄位分類 | 籌碼 |
| 代碼 | `Stotalbuy` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、還原日、還原週、還原月、季、半年、年 |
| 支援商品 | 台股 |

## 語法
> 投信買進張數。  

```pascal
Value1 = GetField("投信買張");
Value1 = GetField("Stotalbuy");
```

---

## 說明
投信買進張數。

週頻率以上為期間累計值。
