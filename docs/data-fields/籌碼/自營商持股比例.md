# 自營商持股比例 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 自營商持股比例 |
| 欄位分類 | 籌碼 |
| 代碼 | `Dsharesheldratio` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、還原日、還原週、還原月、季、半年、年 |
| 支援商品 | 台股 |

## 語法
> 系統估算值，計算公式為：自營商持股/流通在外張數 * 100%。  
> **單位為%，例如：持股比例若為20.5%，則回傳20.5。**  

```pascal
Value1 = GetField("自營商持股比例");
Value1 = GetField("Dsharesheldratio");
```

---

## 說明
系統估算值，計算公式為：自營商持股/流通在外張數 * 100%。

單位為%，例如：持股比例若為20.5%，則回傳20.5。

週頻率以上為統計期間的期底值。
