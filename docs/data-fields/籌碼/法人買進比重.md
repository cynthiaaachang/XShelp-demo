# 法人買進比重 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 法人買進比重 |
| 欄位分類 | 籌碼 |
| 代碼 | `InvestorLongRatio` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 大盤 |

## 語法
> 法人買進金額占大盤成交量的比重。  
> **單位是％，例如買進比重為12%的話會回傳12。**  

```pascal
Value1 = GetField("法人買進比重");
Value1 = GetField("InvestorLongRatio");
```

---

## 說明
法人買進金額占大盤成交量的比重。
單位是％，例如買進比重為12%的話會回傳12。

計算公式：法人買進金額／大盤成交金額＊100%。

週頻率以上為期間買進金額加總／期間大盤成交金額加總。
