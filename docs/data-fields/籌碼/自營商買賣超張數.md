# 自營商買賣超張數 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 自營商買賣超張數 |
| 欄位分類 | 籌碼 |
| 代碼 | `DdifferenceonStock` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、還原日、還原週、還原月、季、半年 |
| 支援商品 | 台股、類股指數 |

## 語法
> 自營商買賣超張數。  

```pascal
Value1 = GetField("自營商買賣超張數");
Value1 = GetField("DdifferenceonStock");
```

---

## 說明
自營商買賣超張數。

台股商品的數值為自營商買張 ─ 自營商賣張。
類股商品（細產業指標、主題指標）的數值則是成分股的自營商買賣超的加總。

週頻率以上為期間累計值。
