# 自營商持股 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 自營商持股 |
| 欄位分類 | 籌碼 |
| 代碼 | `Dsharesheld`<br>`自營商持股張數` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、還原日、還原週、還原月、季、半年、年 |
| 支援商品 | 台股、類股指數 |

## 語法
> 系統估算值，計算方式是累計長期的自營商買賣超張數。  

```pascal
Value1 = GetField("自營商持股");
Value1 = GetField("Dsharesheld");
Value1 = GetField("自營商持股張數");
```

---

## 說明
系統估計值，計算方式是累計長期的自營商買賣超張數。

類股商品（細產業指標、主題指標）的數值則是成分股的自營商持股的加總。

週頻率以上為統計期間的期底值。
