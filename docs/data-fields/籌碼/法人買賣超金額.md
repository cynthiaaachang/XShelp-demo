# 法人買賣超金額 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 法人買賣超金額 |
| 欄位分類 | 籌碼 |
| 代碼 | `InvestorDifferenceAmount` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、季、半年、年 |
| 支援商品 | 大盤 |

## 語法
> 三大法人(外資、投信、自營商)合計買賣超金額。  

```pascal
Value1 = GetField("法人買賣超金額");
Value1 = GetField("InvestorDifferenceAmount");
```

---

## 說明
三大法人(外資、投信、自營商)合計買賣超金額。
計算公式：外資買賣超金額＋投信買賣超金額＋自營商買賣超金額。

週頻率以上為期間累計值。
