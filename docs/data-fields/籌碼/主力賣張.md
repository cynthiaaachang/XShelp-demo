# 主力賣張 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 主力賣張 |
| 欄位分類 | 籌碼 |
| 代碼 | `Leadertotalsell` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、還原日、還原週、還原月、季、半年、年 |
| 支援商品 | 台股 |

## 語法
> 前十五大賣超分點的賣超張數加總。  

```pascal
Value1 = GetField("主力賣張");
Value1 = GetField("Leadertotalsell");
```

---

## 說明
前十五大賣超分點的賣超張數加總。

系統每日統計券商分點對個股的買賣記錄，依照賣超張數(賣出張數 - 買進張數)排序，找出前15大分點，然後把這15個分點對這檔個股的賣超張數加總，計算出主力賣張這個數值。

週頻率以上為期間累計值。
