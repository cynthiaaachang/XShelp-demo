# 控盤者買賣超張數 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 控盤者買賣超張數 |
| 欄位分類 | 籌碼 |
| 代碼 | `Controllerdifference` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、還原日、還原週、還原月、季、半年、年 |
| 支援商品 | 台股 |

## 語法
> 控盤者買張 ─ 控盤者賣張。  

```pascal
Value1 = GetField("控盤者買賣超張數");
Value1 = GetField("Controllerdifference");
```

---

## 說明
控盤者買張 ─ 控盤者賣張。

控盤者買張是單一價格成交值大於100萬之間的合計買進張數，而控盤者賣張則是同樣級距的合計賣出張數。

週頻率以上為期間累計值。
