# 當日沖銷張數 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 當日沖銷張數 |
| 欄位分類 | 籌碼 |
| 代碼 | `Daytradeshares` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、還原日、還原週、還原月、季、半年、年 |
| 支援商品 | 台股、大盤 |

## 語法
> 當日沖銷的總張數。計算公式：現股當沖張數＋資券互抵張數。  

```pascal
Value1 = GetField("當日沖銷張數");
Value1 = GetField("Daytradeshares");
```

---

## 說明
當日沖銷的總張數。計算公式：現股當沖張數＋資券互抵張數。

大盤商品以及類股商品（細產業指標、主題指標）的數值為成分股當日沖銷張數的加總。

週頻率以上為期間累計值。
