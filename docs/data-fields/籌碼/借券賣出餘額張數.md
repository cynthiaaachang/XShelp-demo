# 借券賣出餘額張數 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 借券賣出餘額張數 |
| 欄位分類 | 籌碼 |
| 代碼 | `SBRemain` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、還原日、還原週、還原月、季、半年、年 |
| 支援商品 | 台股、大盤 |

## 語法
> 借券賣出餘額張數，也就是到目前為止借券賣出的淨張數。  

```pascal
Value1 = GetField("借券賣出餘額張數");
Value1 = GetField("SBRemain");
```

---

## 說明
借券餘額張數，也就是到目前為止借券的淨張數。

借券賣出餘額約等於前一日的借券賣出餘額＋當日的借券賣出張數＋當日的借券賣出還券張數。

請注意：上面公式內會加上「借券賣出還券張數」的原因是因為這欄位是負值。

週頻率以上為統計期間的期底值。
