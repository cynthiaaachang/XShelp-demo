# 市場總分點數 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 市場總分點數 |
| 欄位分類 | 籌碼 |
| 代碼 | `TotalBranches` |
| 單位 | 家 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股 |

## 語法
> 市場所有的券商分點家數。  

```pascal
Value1 = GetField("市場總分點數", "D");
Value1 = GetField("TotalBranches", "D");
```

---

## 說明
市場所有的券商分點家數。

包含所有有經營經紀業務的券商分點以及總公司。

這個數值是系統統計交易「分公司」相關欄位(例如分公司交易家數，分公司買進家數，分公司賣出家數等)的母體總數量。
