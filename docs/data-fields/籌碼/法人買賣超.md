# 法人買賣超 – （資料欄位） <kbd>籌碼</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 法人買賣超 |
| 欄位分類 | 籌碼 |
| 代碼 | `InvestorDifferenceTotal` |
| 單位 | 台股,類股: 張. 大盤: 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月、還原日、還原週、還原月、季、半年、年 |
| 支援商品 | 台股、大盤、類股指數 |

## 語法
> 三大法人買賣超。數值內容/單位依商品類型而異。  

```pascal
Value1 = GetField("法人買賣超");
Value1 = GetField("InvestorDifferenceTotal");
```

---

## 說明
三大法人買賣超。數值內容/單位依商品類型而異。

台股商品的數值為三大法人合計買賣超張數。
類股商品（細產業指標、主題指標）的數值則是成分股的三大法人買賣超張數的加總。
大盤商品的數值則是三大法人合計買賣超金額，單位是元。

週頻率以上為期間累計值。
