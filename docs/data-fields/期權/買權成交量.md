# 買權成交量 – （資料欄位） <kbd>期權</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 買權成交量 |
| 欄位分類 | 期權 |
| 代碼 | `CallTotalVolume` |
| 單位 | 口 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日 |
| 支援商品 | 選擇權 |

## 語法
> 開盤到現在的買權成交量總和。  
> 夜盤商品不納入計算。  

```pascal
Value1 = GetField("買權成交量", "D");
Value1 = GetField("CallTotalVolume", "D");
```

---

## 說明
開盤到現在的買權成交量總和。
夜盤商品不納入計算。
