# 波動性指數 – （資料欄位） <kbd>期權</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 波動性指數 |
| 欄位分類 | 期權 |
| 代碼 | `VolaIndex` |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 期貨、選擇權 |

## 語法
> 期交所所公布的「臺指選擇權波動率指數」。  

```pascal
Value1 = GetField("波動性指數");
Value1 = GetField("VolaIndex");
```

---

## 說明
期交所所公布的「臺指選擇權波動率指數」。

此指標是期交所以CBOE VIX指數編製公式計算出的當月臺指選擇權的波動率指數。

亦可使用GetSymbolField("VIX.TF", "Close")來取得最新的數值。
