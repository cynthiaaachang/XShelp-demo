# 估計除息點數 – （資料欄位） <kbd>期權</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 估計除息點數 |
| 欄位分類 | 期權 |
| 代碼 | `EstimatedExDividendPoint` |
| 單位 | 點 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 期貨 |

## 語法
> 受成分股內除息所影響的估計點數。  

```pascal
Value1 = GetField("估計除息點數");
Value1 = GetField("EstimatedExDividendPoint");
```

---

## 說明
受成分股內除息所影響的估計點數。

計算方式是加總從今日到到期日之間，所有成分股的除息所影響的點數。影響點數的計算方式是現金股利／當日收盤價＊商品佔指數的比重。
