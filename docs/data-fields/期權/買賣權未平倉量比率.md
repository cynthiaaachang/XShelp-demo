# 買賣權未平倉量比率 – （資料欄位） <kbd>期權</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 買賣權未平倉量比率 |
| 欄位分類 | 期權 |
| 代碼 | `CPOIRatio` |
| 單位 | 比值 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、週、月 |
| 支援商品 | 選擇權 |

## 語法
> 同契約所有賣權商品(PUT)的未平倉量／買權商品(CALL)的未平倉量。  

```pascal
Value1 = GetField("買賣權未平倉量比率");
Value1 = GetField("CPOIRatio");
```

---

## 說明
同契約所有賣權商品(PUT)的未平倉量／買權商品(CALL)的未平倉量。

統計範圍包含所有履約價/月份(含週契約)。
