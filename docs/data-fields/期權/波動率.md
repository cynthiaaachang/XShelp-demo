# 波動率 – （資料欄位） <kbd>期權</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 波動率 |
| 欄位分類 | 期權 |
| 代碼 | `Volatility` |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 分鐘、日、週、月、還原日、還原週、還原月、季、半年、年 |
| 支援商品 | 台股、大盤、類股指數、台(權證)、香港股票 |

## 語法
> 選擇權／權證商品標的股的歷史波動率。  
> **單位是％，例如數值如果是23.12％，則回傳23.12。**  

```pascal
Value1 = GetField("波動率");
Value1 = GetField("Volatility");
```

---

## 說明
選擇權／權證商品標的股的歷史波動率。
單位是％，例如數值如果是23.12％，則回傳23.12。

計算方式是使用最近20天的股價變化，算出年化的歷史波動率，以此來衡量標的股股價的波動幅度。

除了選擇權／權證商品之外，有發行衍生性商品的台股或是指數商品也都支援此欄位。

週頻率以上為統計期間的期底值。
