# 均價 – （資料欄位） <kbd>常用</kbd> <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 均價 |
| 欄位分類 | 常用、價格 |
| 代碼 | `AvgPrice` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 分鐘、日 |
| 支援商品 | 台股、大盤、類股指數、期貨、選擇權、香港股票、大陸股票 |

## 語法
> 每日的平均成交價。  

```pascal
Value1 = GetField("均價");
Value1 = GetField("AvgPrice");
```

---

## 說明
每日的平均成交價。

計算方式是累加每日開盤後每筆成交的價格乘上數量，然後除以累計的成交數量，得出當日平均的成交價。

如果是分鐘頻率的話，數值為開盤到現在為止的成交均價。如果是日頻率的話則是當日的成交均價。
