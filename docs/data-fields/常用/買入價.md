# 買入價 – （資料欄位） <kbd>常用</kbd> <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 買入價 |
| 欄位分類 | 常用、價格 |
| 代碼 | `BidPrice` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 1 |
| 支援商品 | 全部 |

## 語法
> 這一筆成交明細內的「買進」價格。  

```pascal
Value1 = GetField("買入價", "Tick");
Value1 = GetField("BidPrice", "Tick");
```

---

## 說明
這一筆成交明細內的「買進」價格。

「買進」價格為成交發生當時五檔委託簿的最佳委買價格。
