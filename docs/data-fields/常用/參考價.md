## 語法
> 當日交易的參考價。一般而言是前一交易日的收盤價，如果股票商品有除權息的話，則是除權息後的參考價。  

```pascal
Value1 = GetField("參考價", "D");
Value1 = GetField("RefPrice", "D");
```

---

## 說明
當日交易的參考價。一般而言是前一交易日的收盤價，如果股票商品有除權息的話，則是除權息後的參考價。

除權息的計算公式如下(實際數值以交易所公佈為主)：
1.除息參考價為股價－股息
2.除權參考價為股價／(1+配股率)
3.同時除權息參考價為 (股價-股息)／(1+配股率)
