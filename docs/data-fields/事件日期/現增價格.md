# 現增價格 – （資料欄位） <kbd>事件日期</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 現增價格 |
| 欄位分類 | 事件日期 |
| 代碼 | `CashIncrementAmount` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 語法
> 最新一期現金增資的發行價格。  

```pascal
Value1 = GetField("現增價格");
Value1 = GetField("CashIncrementAmount");
```

---

## 說明
最新一期現金增資的發行價格。

可以檢查GetFieldDate("現增價格")是否等於0來判斷是否有現金增資的資料。
