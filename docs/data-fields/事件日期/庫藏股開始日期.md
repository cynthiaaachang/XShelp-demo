# 庫藏股開始日期 – （資料欄位） <kbd>事件日期</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 庫藏股開始日期 |
| 欄位分類 | 事件日期 |
| 代碼 | `Stockstartdate` |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 語法
> 最新一次庫藏股預計買回期間的開始日期。  
> 欄位格式為西元年月日，例如20221101。  

```pascal
Value1 = GetField("庫藏股開始日期");
Value1 = GetField("Stockstartdate");
```

---

## 說明
最新一次庫藏股預計買回期間的開始日期。

欄位格式為西元年月日，例如20221101。

可以檢查GetFieldDate("庫藏股開始日期")是否等於0來判斷是否有資料。
