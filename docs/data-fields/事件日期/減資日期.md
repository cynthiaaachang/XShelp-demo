# 減資日期 – （資料欄位） <kbd>事件日期</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 減資日期 |
| 欄位分類 | 事件日期 |
| 代碼 | `CapitalReductionDate` |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 語法
> 最近一次減資日期。  
> 欄位格式為西元年月日，例如20221101。  

```pascal
Value1 = GetField("減資日期");
Value1 = GetField("CapitalReductionDate");
```

---

## 說明
最近一次減資日期日期。

欄位格式為西元年月日，例如20221101。

可以檢查GetFieldDate("減資日期")是否等於0來判斷是否有減資的資料。
