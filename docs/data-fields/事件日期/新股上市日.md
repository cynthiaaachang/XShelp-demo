# 新股上市日 – （資料欄位） <kbd>事件日期</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 新股上市日 |
| 欄位分類 | 事件日期 |
| 代碼 | `LaunchDayOfNewShares` |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 語法
> 最近一次新股上市日期。  
> 欄位格式為西元年月日，例如20221101。  

```pascal
Value1 = GetField("新股上市日");
Value1 = GetField("LaunchDayOfNewShares");
```

---

## 說明
最近一次新股上市日期。

欄位格式為西元年月日，例如20221101。

系統依照最近一次公布的股東會，除權，現增，或是減資，回傳最接近的新股上市日。
