# 強弱指標 – （資料欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 強弱指標 |
| 欄位分類 | 價格 |
| 代碼 | `StrongWeak` |
| 單位 | % |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股、類股指數 |

## 語法
> 商品漲跌幅與對應大盤漲跌幅的差異，用來衡量商品是否比大盤強。  

```pascal
Value1 = GetField("強弱指標", "D");
Value1 = GetField("StrongWeak", "D");
```

---

## 說明
商品漲跌幅與對應大盤漲跌幅的差異，用來衡量商品是否比大盤強。

強弱指標的計算公式是(商品漲跌幅－對應大盤漲跌幅)。單位是％，例如商品漲幅是2.5％，對應大盤漲幅是1.8%，那麼強弱指標則為+0.7。
