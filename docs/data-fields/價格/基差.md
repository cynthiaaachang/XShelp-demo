# 基差 – （資料欄位） <kbd>價格</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 基差 |
| 欄位分類 | 價格 |
| 代碼 | `Basis` |
| 單位 | 點 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 分鐘、日、週、月 |
| 支援商品 | 大盤、期貨 |

## 語法
> 現貨價格與期貨價格之間的差異。  

```pascal
Value1 = GetField("基差");
Value1 = GetField("Basis");
```

---

## 說明
現貨價格與期貨價格之間的差異。

例如：加權指數為16800，而大台指為16795的話，那麼基差＝16800 ─ 16795＝5。
