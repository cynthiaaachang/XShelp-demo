| 項目 | 說明 |

| 欄位名稱 | 內外盤 |
| 欄位分類 | 價格 |
| 代碼 | `BidAskFlag` |

## 語法
> 這一筆成交明細內的「內外盤」標記。  
> 數值是1時表示是外盤成交，數值是-1時為內盤成交，如果是0的話則表示無法判斷。  

```pascal
Value1 = GetField("內外盤", "Tick");
Value1 = GetField("BidAskFlag", "Tick");
```

---

## 說明
這一筆成交明細內的「內外盤」標記。 

數值是1時表示是外盤成交，數值是-1時為內盤成交，如果是0的話則表示無法判斷。

系統會依照成交發生當時五檔委託簿的最佳委買價格以及最佳委賣價格來判斷此筆是外盤成交或是內盤成交。

如果成交的價格貼近委賣價格的話，則標示為外盤成交，一般會視為主動「買進」。如果成交的價格貼近委買價格的話，則標示為內盤成交，一般會視為主動「賣出」。
