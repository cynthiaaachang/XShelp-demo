# 內盤均量 – （資料欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 內盤均量 |
| 欄位分類 | 量能 |
| 代碼 | `InAvgVolume` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日、還原日 |
| 支援商品 | 台股 |

## 語法
> 計算公式：內盤成交量／內盤成交次數。  

```pascal
Value1 = GetField("內盤均量");
Value1 = GetField("InAvgVolume");
```
