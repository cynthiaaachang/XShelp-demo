# 開盤買筆 – （資料欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 開盤買筆 |
| 欄位分類 | 量能 |
| 代碼 | `OpenBidTx` |
| 單位 | 筆 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日 |
| 支援商品 | 大盤、期貨、選擇權 |

## 語法
> 開盤時委託買進的筆數。  

```pascal
Value1 = GetField("開盤買筆", "D");
Value1 = GetField("OpenBidTx", "D");
```

---

## 說明
開盤時委託買進的筆數。

大盤的數值會扣除權證。
