# 上漲量 – （資料欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 上漲量 |
| 欄位分類 | 量能 |
| 代碼 | `UpVolume` |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 分鐘、日 |
| 支援商品 | 台股、大盤、類股指數、期貨、選擇權 |

## 語法
> 這一筆K棒的所有成交內比前一價上漲的成交量的加總。  

```pascal
Value1 = GetField("上漲量");
Value1 = GetField("UpVolume");
```

---

## 說明
這一筆K棒的所有成交內比前一價上漲的成交量的加總。

例如，這一筆K棒內有4筆成交，分別是30元2張，30元3張，31元4張，30元5張，而前一筆K棒的收盤價是29。那麼上漲量=6張，下跌量=5張。

計算方法如下：

- 第一筆成交30元比前一價29元上漲(上漲量=2張)
- 第二筆成交30元跟前一價30元相同，不予計算
- 第三筆31元比前一價30元上漲(上漲量=2張+4張=6張)
- 第四筆30元比前一價31元下跌(下跌量=5張)
