# 累委賣筆 – （資料欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 累委賣筆 |
| 欄位分類 | 量能 |
| 代碼 | `TotalAskTx` |
| 單位 | 筆 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 分鐘、日 |
| 支援商品 | 大盤、期貨、選擇權 |

## 語法
> 開盤迄今的總委賣筆數。  

```pascal
Value1 = GetField("累委賣筆");
Value1 = GetField("TotalAskTx");
```

---

## 說明
開盤迄今的總委賣筆數。

交易所盤中會定時公布此資訊。如果是分鐘頻率的話會取得當下最新數值，如果是日頻率的話則是當日最後的數值。

證券市場的數值為扣除權證的統計。
