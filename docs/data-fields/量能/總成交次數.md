# 總成交次數 – （資料欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 總成交次數 |
| 欄位分類 | 量能 |
| 代碼 | `TotalTicks` |
| 單位 | 筆 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 分鐘、日 |
| 支援商品 | 台股、期貨、選擇權、香港股票、大陸股票、美(股票) |

## 語法
> 成交的總筆數。  

```pascal
Value1 = GetField("總成交次數");
Value1 = GetField("TotalTicks");
```

---

## 說明
成交的總筆數。

資料來源為交易所盤中所傳送的成交明細的筆數。

如果分鐘頻率的話則為該分鐘統計區間內的成交明細筆數。
