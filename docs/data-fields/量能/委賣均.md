# 委賣均 – （資料欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 委賣均 |
| 欄位分類 | 量能 |
| 代碼 | `AvgShortShare`<br>`AvgShortUnits`<br>`均賣張數`<br>`均賣口數` |
| 單位 | 大盤:張；期貨/選擇權:口 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 日 |
| 支援商品 | 大盤、期貨、選擇權 |

## 語法
> 每一筆委賣的平均張數/口數。  
> 計算公式：累計委賣／累委賣筆。  

```pascal
Value1 = GetField("委賣均", "D");
Value1 = GetField("AvgShortShare", "D");
Value1 = GetField("AvgShortUnits", "D");
Value1 = GetField("均賣張數", "D");
Value1 = GetField("均賣口數", "D");
```

---

## 說明
每一筆委賣的平均張數/口數。
計算公式：累計委賣／累委賣筆。
