# 量比 – （資料欄位） <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 量比 |
| 欄位分類 | 量能 |
| 代碼 | `VolumeRatio` |
| 單位 | 比值 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、函數 |
| 可用頻率 | 分鐘、日 |
| 支援商品 | 類股指數、台(股票)、香港股票、大陸股票 |

## 語法
> 當日估計量與昨量的放大比例。  
> 計算公式：估計量／昨量。  

```pascal
Value1 = GetField("量比");
Value1 = GetField("VolumeRatio");
```

---

## 說明
當日估計量與昨量的放大比例。

計算公式：估計量／昨量。

這個數值越大表示當日的成交量可能會放大，可以當成是交易熱絡度的一個指標。
