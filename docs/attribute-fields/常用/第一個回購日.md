# 第一個回購日 – （屬性欄位） <kbd>常用</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 第一個回購日 |
| 欄位分類 | 常用 |
| 代碼 | `FirstCallDate` |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 16.US.P |

## 語法
> **回傳美股特別股的第一個回購日期。**  
> 欄位格式為西元年月日，例如20221101。  

```pascal
Value1 = GetField("第一個回購日");
Value1 = GetField("FirstCallDate");
```
