# ETD – （屬性欄位） <kbd>常用</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | ETD |
| 欄位分類 | 常用 |
| 代碼 | `ExchangeTradedDebt` |
| 格式 | 布林值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 16.US.P |

## 語法
> 是否是交易所交易債券(ETD)。格式是布林值。  

```pascal
Value1 = GetField("ETD");
Value1 = GetField("ExchangeTradedDebt");
```
