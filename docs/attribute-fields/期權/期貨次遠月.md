# 期貨次遠月 – （屬性欄位） <kbd>期權</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 期貨次遠月 |
| 欄位分類 | 期權 |
| 代碼 | `Future*3` |
| 格式 | 字串 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 期貨 |

## 語法
> **回傳商品的相關期貨次遠月商品代碼。格式是字串。**  
> 例如FITXN03.TF。  

```pascal
Value1 = GetField("期貨次遠月");
Value1 = GetField("Future*3");
```
