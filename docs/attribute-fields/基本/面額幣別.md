# 面額幣別 – （屬性欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 面額幣別 |
| 欄位分類 | 基本 |
| 代碼 | `ParValueCurrency` |
| 格式 | 字串 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 台股 |

## 語法
> **回傳股票的發行幣別。格式是字串，例如"台幣"，"美元"等。**  

```pascal
Value1 = GetField("面額幣別");
Value1 = GetField("ParValueCurrency");
```
