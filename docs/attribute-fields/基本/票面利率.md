# 票面利率 – （屬性欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 票面利率 |
| 欄位分類 | 基本 |
| 代碼 | `CouponRate` |
| 單位 | % |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 1.B、1.P、16.US.P |

## 語法
> **回傳可轉債商品(CB)或是特別股的票面利率。格式是數值。**  

```pascal
Value1 = GetField("票面利率");
Value1 = GetField("CouponRate");
```
