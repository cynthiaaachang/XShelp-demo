# 發行張數 – （屬性欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 發行張數 |
| 欄位分類 | 基本 |
| 代碼 | `IssueAmount` |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 1.B |

## 語法
> **回傳可轉債商品(CB)的發行張數。**  

```pascal
Value1 = GetField("發行張數");
Value1 = GetField("IssueAmount");
```
