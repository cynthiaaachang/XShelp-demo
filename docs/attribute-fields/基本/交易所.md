# 交易所 – （屬性欄位） <kbd>基本</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 交易所 |
| 欄位分類 | 基本 |
| 代碼 | `Exchange` |
| 格式 | 字串 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 台股、台(權證)、1.B、1.P、美(股票) |

## 語法
> **回傳商品掛牌的交易所。格式是字串。**  

```pascal
Value1 = GetField("交易所");
Value1 = GetField("Exchange");
```

---

## 說明
回傳商品掛牌的交易所。格式是字串。

如果是台股的話，上市股票回傳TSE、上櫃股票則回傳OTC。
如果是美股的話，則回傳NYSE、NASDAQ、AMEX。
