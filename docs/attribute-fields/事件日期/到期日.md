# 到期日 – （屬性欄位） <kbd>事件日期</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 到期日 |
| 欄位分類 | 事件日期 |
| 代碼 | `ExpirationDate` |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 台(權證)、1.B、期貨、選擇權、16.US.P |

## 語法
> **回傳商品的到期日。**  
> 欄位格式為西元年月日，例如20221101。  

```pascal
Value1 = GetField("到期日");
Value1 = GetField("ExpirationDate");
```
