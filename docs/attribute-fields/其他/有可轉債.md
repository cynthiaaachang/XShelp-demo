# 有可轉債 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 有可轉債 |
| 欄位分類 | 其他 |
| 代碼 | `WithConvertibleBond` |
| 格式 | 布林值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 台股 |

## 語法
> **回傳商品是否有發行可轉債(CB)商品。格式是布林值。**  

```pascal
Value1 = GetField("有可轉債");
Value1 = GetField("WithConvertibleBond");
```
