# 賣回權日期 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 賣回權日期 |
| 欄位分類 | 其他 |
| 代碼 | `PutbackDate` |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 1.B |

## 語法
> 指的是可轉債發行公司設置的某個特定日期，在此日期，投資人可以選擇是否行使賣回權。  

```pascal
Value1 = GetField("賣回權日期");
Value1 = GetField("PutbackDate");
```
