# 有熊證 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 有熊證 |
| 欄位分類 | 其他 |
| 代碼 | `WithBearWarrant` |
| 格式 | 布林值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 台股、大盤 |

## 語法
> **回傳商品是否有發行熊證權證商品。格式是布林值。**  

```pascal
Value1 = GetField("有熊證");
Value1 = GetField("WithBearWarrant");
```
