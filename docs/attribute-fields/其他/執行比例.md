# 執行比例 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 執行比例 |
| 欄位分類 | 其他 |
| 代碼 | `ExecutionRatio` |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 台(權證) |

## 語法
> **回傳權證商品的執行比例。格式是數值。**  

```pascal
Value1 = GetField("執行比例");
Value1 = GetField("ExecutionRatio");
```
