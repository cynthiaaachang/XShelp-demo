# 處置股 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 處置股 |
| 欄位分類 | 其他 |
| 代碼 | `IsDisposition`<br>`警示股` |
| 格式 | 布林值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 台股、台(權證)、1.B、1.P |

## 語法
> **回傳當日是否為處置股(警示股)。格式是布林值。**  

```pascal
Value1 = GetField("處置股");
Value1 = GetField("IsDisposition");
Value1 = GetField("警示股");
```

---

回傳當日是否為處置股(警示股)。格式是布林值。
