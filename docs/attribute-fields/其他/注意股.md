# 注意股 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 注意股 |
| 欄位分類 | 其他 |
| 代碼 | `IsAttention` |
| 格式 | 布林值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 台股、台(權證)、1.B、1.P |

## 語法
> **回傳當日是否為注意股。格式是布林值。**  

```pascal
Value1 = GetField("注意股");
Value1 = GetField("IsAttention");
```
