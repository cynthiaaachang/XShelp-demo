# 買賣權 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 買賣權 |
| 欄位分類 | 其他 |
| 代碼 | `CallOrPut` |
| 格式 | 字串 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 台(權證)、選擇權 |

## 語法
> **回傳選擇權商品或是權證商品的買賣權別。格式是字串(CALL/PUT)。**  

```pascal
Value1 = GetField("買賣權");
Value1 = GetField("CallOrPut");
```
