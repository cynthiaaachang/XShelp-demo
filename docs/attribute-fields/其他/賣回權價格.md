# 賣回權價格 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 賣回權價格 |
| 欄位分類 | 其他 |
| 代碼 | `PutbackPrice` |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 1.B |

## 語法
> 是指投資人在賣回權日期時，如果選擇行使賣回權，可以將可轉債賣回給發行公司時的價格。  

```pascal
Value1 = GetField("賣回權價格");
Value1 = GetField("PutbackPrice");
```
