# 累計異常注意股 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 累計異常注意股 |
| 欄位分類 | 其他 |
| 代碼 | `SumAttention` |
| 格式 | 布林值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |

## 語法

```pascal

Value1 = GetField("SumAttention");
```
