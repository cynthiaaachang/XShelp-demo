# 可轉換日 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 可轉換日 |
| 欄位分類 | 其他 |
| 代碼 | `ConversionDate` |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 1.B |

## 語法
> **回傳可轉債商品(CB)的可轉換日期。**  
> 欄位格式為西元年月日，例如20221101。  

```pascal
Value1 = GetField("可轉換日");
Value1 = GetField("ConversionDate");
```
