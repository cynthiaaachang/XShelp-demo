# 先買現沖 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 先買現沖 |
| 欄位分類 | 其他 |
| 代碼 | `IsBuyDayTrading` |
| 格式 | 布林值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 台股 |

## 語法
> **回傳當日是否可以現股先買後賣。格式是布林值。**  

```pascal
Value1 = GetField("先買現沖");
Value1 = GetField("IsBuyDayTrading");
```
