# 標的物 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 標的物 |
| 欄位分類 | 其他 |
| 代碼 | `Underlying` |
| 格式 | 字串 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 台(權證)、1.B、1.P、期貨、選擇權 |

## 語法
> **回傳衍生性商品所對應的標的商品代碼。格式是字串。**  

```pascal
Value1 = GetField("標的物");
Value1 = GetField("Underlying");
```
