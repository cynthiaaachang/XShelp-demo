# 面額 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 面額 |
| 欄位分類 | 其他 |
| 代碼 | `ParValue` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 台股 |

## 語法
> **回傳股票每一股的面額(是多少元)。格式是數值。**  

```pascal
Value1 = GetField("面額");
Value1 = GetField("ParValue");
```
