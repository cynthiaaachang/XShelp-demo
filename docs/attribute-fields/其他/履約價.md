# 履約價 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 履約價 |
| 欄位分類 | 其他 |
| 代碼 | `StrikePrice` |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 台(權證)、選擇權 |

## 語法
> **回傳選擇權商品或是權證商品的履約價。格式是數值。**  

```pascal
Value1 = GetField("履約價");
Value1 = GetField("StrikePrice");
```

---

## 說明
回傳選擇權商品或是權證商品的履約價。格式是數值。
