# 轉換價格 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 轉換價格 |
| 欄位分類 | 其他 |
| 代碼 | `ConversionPrice` |
| 格式 | 數值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 1.B |

## 語法
> **回傳可轉債商品(CB)最新的轉換價格。格式是數值。**  

```pascal
Value1 = GetField("轉換價格");
Value1 = GetField("ConversionPrice");
```
