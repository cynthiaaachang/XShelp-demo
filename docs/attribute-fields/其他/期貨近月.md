# 期貨近月 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 期貨近月 |
| 欄位分類 | 其他 |
| 代碼 | `Future*1` |
| 格式 | 字串 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 台股、期貨 |

## 語法
> **回傳商品的相關期貨近月商品代碼。格式是字串。**  
> 例如FITXN01.TF、FICDF01.TF。  

```pascal
Value1 = GetField("期貨近月");
Value1 = GetField("Future*1");
```
