# 可放空 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 可放空 |
| 欄位分類 | 其他 |
| 代碼 | `ShortSellStock` |
| 格式 | 布林值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 台股 |

## 語法
> **回傳當日是否可以融券放空。格式是布林值。**  

```pascal
Value1 = GetField("可放空");
Value1 = GetField("ShortSellStock");
```
