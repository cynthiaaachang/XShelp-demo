# 擔保品 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 擔保品 |
| 欄位分類 | 其他 |
| 代碼 | `SecuredCB` |
| 格式 | 布林值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 1.B |

## 語法
> **回傳可轉換商品(CB)是否有擔保。格式是布林值。**  

```pascal
Value1 = GetField("擔保品");
Value1 = GetField("SecuredCB");
```
