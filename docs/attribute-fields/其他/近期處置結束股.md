# 近期處置結束股 – （屬性欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 近期處置結束股 |
| 欄位分類 | 其他 |
| 代碼 | `IsRecentlyEndDisposition` |
| 格式 | 布林值 |
| 支援腳本 | 指標、警示、交易、選股、函數 |
| 支援商品 | 台股、台(權證)、1.B |

## 語法
> **回傳商品是否在過去7日內剛結束處置。格式是布林值。**  

```pascal
Value1 = GetField("近期處置結束股");
Value1 = GetField("IsRecentlyEndDisposition");
```
