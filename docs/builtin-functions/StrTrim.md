## StrTrim（去除字串開頭及結尾的空白後回傳字串）

**回傳類型**: 1\
**參數數量**: 3

### 說明

str1 = StrTrim(字串); str1 = StrTrim(字串, 選項);

-   傳入字串後，去除這個字串開頭以及結尾的空白後回傳字串。
-   預設是刪除空白。
-   可以傳入第二個參數選項，指定刪除的範圍。
-   選項是一個數字，0 = 頭尾都刪除(如同預設)， 1 = 只去除開頭的空白，2 =
    只去除結尾的空白。

### 參數

-   \*\*\*\*（type: 3）
-   \*\*\*\*（type: 3）
-   \*\*\*\*（type: 2）
