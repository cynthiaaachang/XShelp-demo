# Round – （內建函數） <kbd>數學函數</kbd>

## 語法

> **回傳數值四捨五入後的結果，可以指定小數位數。**  
> **回傳數值 = Round(數值，小數位數)**  

---

## 說明

範例:

```pascal
Value1 = Round(10.547, 0); // Value1 = 11
Value2 = Round(10.547, 1); // Value1 = 10.5
Value3 = Round(10.547, 2); // Value1 = 10.55
```
