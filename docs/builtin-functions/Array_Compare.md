## Array_Compare（比較陣列內元素）

**回傳類型**: 1\
**參數數量**: 5

### 說明

比較陣列A跟陣列B內的元素 Array_Compare(陣列A, 陣列A開始比對的位置,
陣列B, 陣列B開始比對的位置, 比對的個數)
如果陣列A比較大則回傳1，如果陣列B比較大則回傳-1，如果相同的話則回傳0，例外情形回傳-2

### 參數

-   **陣列1**（type: 13;14;15）
-   **元素位置1**（type: 1）
-   **陣列2**（type: 13;14;15）
-   **元素位置2**（type: 1）
-   **長度**（type: 1）
