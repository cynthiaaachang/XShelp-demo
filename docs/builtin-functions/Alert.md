# Alert（內建函數）— 交易函數

## 語法

> 在自動交易腳本中執行提醒訊息產生一筆警示紀錄  
> **Alert(字串1)**  
> **Alert(字串1, 數值1, 數值2 …)**

---

## 說明

`Alert` 函數可用來輸入一個或多個參數，參數的型別可以是文字、數字、或是有價格，系統會自動把輸入的多參數轉成字串格式。  
如果傳入多個參數，系統會自動在參數之間加入空白分隔。  

當策略觸發 Alert 時，警示內容會出現在執行紀錄、商品監控、提示視窗以及手機推播通知中，方便使用者快速了解觸發原因。
