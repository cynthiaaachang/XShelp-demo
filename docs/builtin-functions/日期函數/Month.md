# Month – （內建函數） <kbd>日期函數</kbd>

## 語法
> 計算傳入日期的月份  
> **月份 = Month(日期)**  
> **回傳月份的範圍從1到12**  

---

## 說明
日期數值是一個8碼的數字，格式為YYYYMMDD:

- YYYY是西元年份，例如2015，
- MM是月份，兩位數字，範圍從01到12，
- DD是日期，兩位數字，範圍從01到31，

日期數值通常是透過CurrentDate，或是Date(資料的日期欄位)，或是其他日期相關函數所產生的日期數值。
回傳的數值則是這個日期的月份，可能的數值從1到12。

舉例：

```pascal
Value1 = Month(20150601);　　// Value1 = 6
```
