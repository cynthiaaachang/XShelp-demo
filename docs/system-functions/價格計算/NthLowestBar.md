# NthLowestBar – （系統函數） <kbd>價格計算</kbd>

## 語法

> 計算序列資料的第N個極小值距當期K棒的相對位置。  
> **回傳數值=NthLowestBar(第幾個極小值,數列,期數)**  
> 傳入三個參數:  
> - 第一個參數是要計算極小值的序號。  
> - 第二個參數是數列，通常是開高低收的價格數列。  
> - 第三個參數是期數。  

---

## 說明

範例：

```pascal
value1 = NthLowestBar(2,low,5); //計算近5期次低的最低價的相對位置
plot1(value1);                  //繪製近5期次低的最低價相對位置的連線
```
