# NthHighestBar – （系統函數） <kbd>價格計算</kbd>

## 語法

> 計算序列資料的第N個極大值距當期K棒的相對位置。  
> **回傳數值=NthHighestBar(第幾個極大值,數列,期數)**  
> 傳入三個參數:  
> - 第一個參數是要計算極大值的序號。  
> - 第二個參數是數列，通常是開高低收的價格數列。  
> - 第三個參數是期數。  

---

## 說明

範例：

```pascal
value1 = NthHighestBar(2,high,5); //計算近5期次高的最高價的相對位置
plot1(value1);                    //繪製近5期次高的最高價相對位置的連線
```
