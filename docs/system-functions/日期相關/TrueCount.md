# TrueCount – （系統函數） <kbd>日期相關</kbd>

## 語法

> 計算符合條件連續符合的次數。  
> **回傳數值=TrueCount(條件數列,期數)**  
> 傳入二個參數:  
> - 第一個參數是條件數列。  
> - 第二個參數是期數。  

---

## 說明

以最新的資料為基準，往前計算連續符合條件的次數。

範例：

```pascal
value1 = TrueCount(close>close[1],5); //計算過去5天連續收紅的次數
```

註：TrueCount函數和[CountIfARow](api.aspx?a=CountIfARow&b=sys)函數相同。
