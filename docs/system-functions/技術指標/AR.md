# AR – （系統函數） <kbd>技術指標</kbd>

## 語法
> 計算人氣指標值，買賣氣勢強度的測試指標，觀察開盤後對股價的現實反應。  
> **回傳數值=AR(期數)**  

---

## 說明
AR值高時，代表行情很活潑，當AR值介於0.25至1.85(25%至185%)間時，屬於盤整行情。
AR值低時，表示人氣不足。

計算公式如下：
 
AR ＝ (今日最高價－今日開盤價) N日內總合 ／ (今日開盤價－今日最低價) N日內總合
 
範例：

```pascal
value1 = AR(26);       //計算26期的AR指標
plot1(value1, "AR");   
```
