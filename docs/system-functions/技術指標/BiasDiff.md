# BiasDiff – （系統函數） <kbd>技術指標</kbd>

## 語法

> 計算長短期乖離率的差值。  
> **回傳數值=BiasDiff(短期期數,長期期數)**  
> 傳入二個參數:  
> - 第一個參數是較短的期數。  
> - 第二個參數是較長的期數。  

---

## 說明

除了價格數列和均線間會出現均線回歸的現象外，長短期的均線也會反復出現發散、收斂的循環現象。乖離率差是用來觀察長短期均線相對關係。

範例：

```pascal
value1 = BiasDiff(3,6);       //計算3期與6期的乖離率差
plot1(value1, "BiasDiff");   
```
