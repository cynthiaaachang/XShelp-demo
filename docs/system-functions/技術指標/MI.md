# MI – （系統函數） <kbd>技術指標</kbd>

## 語法
> 計算質量指標。  
> **回傳數值=MI(期數1,期數2)**  
> 傳入二個參數:  
> - 第一個參數是計算指數移動平均的期數。  
> - 第二個參數是計算總和的期數。  

---

## 說明
MI指標（Mass Index）是由Donald Dorsey所發明。質量指標是觀察高低價範圍的變化，試圖找出趨勢的反轉點。

計算公式： 

EMA1 = (最高價－最低價)的9日EMA
EMA2 = EMA1 的9日EMA
MI = ( EMA1 ／ EMA2 ) 的N2日簡單加總

範例：

```pascal
value1 = MI(9,25);       //計算收盤價10期的質量指標
plot1(value1, "MI");   
```
