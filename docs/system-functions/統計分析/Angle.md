# Angle – （系統函數） <kbd>統計分析</kbd>

## 語法
> 計算任意二個日期的走勢角度。  
> **回傳數值=Angle(日期1,日期2)**  
> 傳入二個參數:  
> - 第一個參數是日期1。  
> - 第二個參數是日期2，需大於日期1。  

---

## 說明
任意二個日期間的價格角度可以代表趨勢的強度。我們利用第一個日期的開盤價及第二個日期的收盤價計算整段走勢的角度。

當回傳數值大於0時，代表趨勢向上；當回傳數值小於0時，代表趨勢向下。

範例：

```pascal
value1 = Angle(date[3],date); //計算近四期走勢的角度
```
