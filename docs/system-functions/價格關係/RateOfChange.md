# RateOfChange – （系統函數） <kbd>價格關係</kbd>

## 語法
> 計算序列資料的變化率。  
> **回傳數值=RateOfChange(數列,期數)**  
> 傳入二個參數:  
> - 第一個參數是數列，通常是開高低收的價格數列。  
> - 第二個參數是期數。  

---

## 說明
以前N根K棒資料為基準點，計算至當根K棒的的數值變化率。

計算公式：

RateOfChange = ( 當期價格 ／ N期前價格 - 1 ) * 100

範例：

```pascal
value1 = RateOfChange(Close,5); //計算5期收盤價的變化率
```
