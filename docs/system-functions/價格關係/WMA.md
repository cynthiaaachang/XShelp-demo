# WMA – （系統函數） <kbd>價格關係</kbd>

## 語法
> 計算序列資料的加權移動平均。  
> **回傳數值=WMA(數列,期數)**  
> 傳入二個參數:  
> - 第一個參數是數列，通常是開高低收的價格數列。  
> - 第二個參數是期數。  

---

## 說明
以最新一筆資料為基準點，輸入要計算的期數，然後計算過去期數的加權移動平均值。

計算公式：

WMAt=(Pt * n + Pt-1 * (n-1) + Pt-2 * (n-2) + ... + Pt-n+1) / (n + (n-1) + (n-2) + ... + 1)

範例：

```pascal
value1 = WMA(Close,5); //計算5期收盤價的加權移動平均
```
