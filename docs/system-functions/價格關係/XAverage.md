# XAverage – （系統函數） <kbd>價格關係</kbd>

## 語法

> 計算序列資料的指數移動平均。  
> **回傳數值=XAverage(數列,期數)**  
> 傳入二個參數:  
> - 第一個參數是數列，通常是開高低收的價格數列。  
> - 第二個參數是期數。  

---

## 說明

以最新一筆資料為基準點，輸入要計算的期數，然後計算過去期數的指數移動平均值。

計算公式：

當期均價 = (2/(n+1))*當期價格 + (n-1)/(n+1)*前期均價

範例：

```pascal
value1 = XAverage(Close,5); //計算5期收盤價的指數移動平均
```
